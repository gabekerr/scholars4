    Section 1.3 The chain rule and tangent spaces

    One of the most important ways one can manipulate a function is by composing it with another function. Indeed, one example of this is changing coordinates from one system to another and we will examine such coordinate systems as we progress. To see how derivatives are effected by this change, we first state some basic lemmas.

    Lemma 1.3.1.

    If \(\mb{H}\) is differentiable at \(\mb{c}\) then there is a neighborhood \(U\) of \(\mb{c}\) and a constant \(C\) such that
    \begin{equation*} \frac{\| \mb{H} (\mb{x} ) - \mb{H} (\mb{c}) \|}{\| \mb{x} - \mb{c} \|} \lt C \end{equation*}
    for all \(\mb{x} \ne \mb{c}\) in \(U\text{.}\)

    Proof.

    This follows from the definition of the derivative, the triangle inequality and continuity of linear transformations.

    Definition 1.3.2.

    Let \(U\) be an open neighborhood of \(\mb{c}\) in an inner product space \(V\) and \(W\) some other inner product space. We say that a function \(E : U \to W\) vanishes higher than first order at \(\mb{c}\) if
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} \frac{E (\mb{x} )}{\| \mb{x} - \mb{c}\|} = \mb{0}. \end{equation*}

    Lemma 1.3.3.

    Suppose \(U_1\) and \(U_2\) are neighborhoods of \(\mb{c}_1\) and \(\mb{c}_2\) in inner product spaces \(V_1\) and \(V_2\) and \(\mb{H} : U_1 \to U_2\) is differentiable at \(\mb{c}_1\) with \(\mb{H} (\mb{c}_1) = \mb{c}_2\text{.}\) If \(E : U_2 \to K\) vanishes higher than first order at \(\mb{c}_2\) then \(E \circ \mb{H}\) vanishes higher than first order at \(\mb{c}_2\text{.}\)

    Proof.

    By definition, for any \(\tilde{\varepsilon} > 0\) there is a \(\delta\) so that if \(\| \mb{y} - \mb{c}_2\| \lt \delta\) then \(\|E (\mb{y} )\| \lt \tilde{\varepsilon} \| \mb{y} - \mb{c}_2\|\text{.}\) Using the continuity of \(\mb{H}\) and Lemma 1.3.1 we have that there is a \(C\) and, in a neighborhood of \(\mb{c}_1\) (which maps to an appropriate neighborhood of \(\mb{c}_2\)),
    \begin{equation*} 0 \leq \frac{\|E ( \mb{H} (\mb{x} ))\|}{\| \mb{x} - \mb{c}_1\|} \lt \tilde{\varepsilon} \frac{\| \mb{H} ( \mb{x} ) - \mb{H} ( \mb{c}_1) \|}{\|\mb{x} - \mb{c}_1\|} \lt \tilde{\varepsilon} C. \end{equation*}
    Taking \(\delta\) for \(\varepsilon / C \) gives the result.
    With these preliminary lemmas in hand, we state the following amazing result.

    Theorem 1.3.4. Chain Rule.

    Let \(V_1, V_2\) and \(V_3\) be finite dimensional real inner product spaces and \(U_1, U_2\) open sets in \(V_1\) and \(V_2\) respectively. Suppose
    \begin{equation*} \mb{G} : U_1 \to U_2 \hspace{.3in} \mb{F} : U_2 \to V_3 \end{equation*}
    are functions for which
    1. \(\mb{G} (\mb{a} ) = \mb{b}\text{,}\)
    2. \(\mb{G}\) is differentiable at \(\mb{a}\text{,}\)
    3. \(\mb{F}\) is differentiable at \(\mb{b}\text{.}\)
    Then \(\mb{F} \circ \mb{G}\) is differentiable at \(\mb{a}\) and
    \begin{equation*} \tder{\mb{a}}{(\mb{F} \circ \mb{G})} = \tder{\mb{b}}{\mb{F}} \circ \tder{\mb{a}}{\mb{G}}. \end{equation*}

    Proof.

    We will use Lemma 1.3.3 and the definition of the vanishing of higher than first order errors. However, the proof is fairly basic up to this result as we now observe. The error terms \(E_F (\mb{y})\) and \(E_G (\mb{x})\) vanish higher than first order at \(\mb{b}\) and \(\mb{a}\) by the definition of the derivative. These are the terms in the equations
    \begin{align*} \mb{F} (\mb{y}) \amp = \mb{F} ( \mb{b} ) + \tder{\mb{b}}{\mb{F}} (\mb{y} - \mb{b} ) + E_F (\mb{y}), \\ \mb{G} (x) \amp = \mb{G} ( \mb{a} ) + \tder{\mb{a}}{\mb{G}} (\mb{x} - \mb{a} ) + E_G (\mb{x}). \end{align*}
    Putting these together gives
    \begin{align*} \mb{F} \circ \mb{G} (\mb{x} ) \amp = \mb{F} (\mb{G} (\mb{x} )), \\ \amp = \mb{F} \left( \mb{b} \right) + \tder{\mb{b}}{\mb{F}} (\mb{G} (\mb{x} ) - \mb{b} ) + E_F (\mb{G} (\mb{x} )), \\ \amp = \mb{F} \left( \mb{G} (\mb{a} ) \right) + \tder{\mb{b}}{\mb{F}} (\tder{\mb{a}}{\mb{G}} (\mb{x} - \mb{a} ) + E_G (\mb{x})) + E_F (\mb{G} (\mb{x} )), \\ \amp = \mb{F} \left( \mb{G} (\mb{a} ) \right) + ( \tder{\mb{b}}{\mb{F}} \circ \tder{\mb{a}}{\mb{G}} ) (\mb{x} - \mb{a} ) + \tder{\mb{b}}{\mb{F}} (E_G (\mb{x})) + E_F (\mb{G} (\mb{x} )). \end{align*}
    Now, the function
    \begin{equation*} \tder{\mb{b}}{\mb{F}} (E_G (\mb{x})) \end{equation*}
    vanishes to higher order since \(\tder{\mb{b}}{\mb{F}}\) is a linear transformation so that
    \begin{equation*} \lim_{\mb{x} \to \mb{a}} \frac{\tder{\mb{b}}{\mb{F}}(E_G (\mb{x}))}{\| \mb{x} - \mb{a} \|} = \lim_{\mb{x} \to \mb{a}}\tder{\mb{b}}{\mb{F}} \left( \frac{E_G (\mb{x})}{\| \mb{x} - \mb{a} \|} \right) = \tder{\mb{b}}{\mb{F}} \lim_{\mb{x} \to \mb{a}} \frac{E_G (\mb{x})}{\| \mb{x} - \mb{a} \|} = \mb{0}. \end{equation*}
    By Lemma 1.3.3, the function \(E_F (\mb{G} (\mb{x} ))\) also vanishes higher than first order. By the definition of the derivative, this gives the result.
    A student that has never understood the chain rule in \(1\)-variable should now see it for what it is : the derivative of a composition is the composition of the derivatives. Of course, to understand this, one must be careful to evaluate these derivative at the correct points (in particular, one must evaluate the derivative of \(\mb{F}\) at the image point \(\mb{G} (\mb{a})\text{.}\)) We now will progress in two ways, an abstract and a computational (with a geometric understanding at the end to tie these together).

    Subsection 1.3.1 Computational considerations

    While our presentation of the chain rule is by far the better way of understanding it, it is rarely given in these terms as most students are not required to understand linear algebra before taking multivariable calculus. Instead, most students learn a few of the basic versions of the chain rule.
    Let us take the case when \(\mb{G}\) is a path \(\mb{G} (t) = ( x(t), y(t))\) and \(\mb{F} = f(x,y)\) is a scalar valued function. Then the derivatives can be computed as Jacobian matrices which are
    \begin{equation*} \tder{}{\mb{G}} = \twovec{\frac{d x}{d t}}{\frac{dy}{dt}}, \end{equation*}
    and
    \begin{equation*} \tder{}{\mb{F}} = \left[ \begin{matrix} \frac{\partial f}{\partial x} \amp \frac{\partial f}{\partial y} \end{matrix} \right]. \end{equation*}
    Composing these and applying the chain rule gives
    \begin{equation*} \frac{d f}{d t} = \frac{\partial f}{\partial x} \frac{d x}{d t} + \frac{\partial f}{\partial y} \frac{d y}{d t}. \end{equation*}

    Example 1.3.5. Chain rule computation I.

    Suppose we parameterize the unit circle as usual with \((\cos t, \sin t)\) and let \(z = f(x,y) = x^2 - y^2\text{.}\) One may ask how \(z\) is changing relative to the parameter \(t\text{.}\) This of course is given by the derivative
    \begin{align*} \frac{dz}{dt} \amp = \frac{\partial z}{\partial x} \frac{d x}{d t} + \frac{\partial z}{\partial y} \frac{d y}{d t}, \\ \amp = \left( 2x \right) (- \sin t) + \left( - 2y \right) \left( \cos t \right) , \\ \amp = \left( 2\cos t \right) (- \sin t) + \left( - 2\sin t \right) \left( \cos t \right), \\ \amp = - 4 \cos t \sin t, \\ \amp = - 2 \sin (2t). \end{align*}
    Note that here we used \(z\) instead of \(f\text{.}\) This is common practice when we view a given variable (like \(z\)) as a dependent parameter.
    The chain rule is often used if one is changing coordinates. In other words, if \(\mb{G} (u, v) = (x ( u, v), y(u, v))\) gives another coordinate system on a region in the plane and \(f(x,y)\) is a scalar function, we may want to compute the change of \(f\) relative to the variables \(u\) and \(v\) instead of \(x\) and \(y\text{.}\) For this, we simply write out the Jacobian matrix of \(\mb{G}\) which is
    \begin{equation*} \tder{}{\mb{G}} = \left[ \begin{matrix} \frac{\partial x}{\partial u} \amp \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial u} \amp \frac{\partial y}{\partial v} \end{matrix} \right] \end{equation*}
    and compose with the Jacobian
    \begin{equation*} \tder{}{f} = \left[ \begin{matrix} \frac{\partial f}{\partial x} \amp \frac{\partial f}{\partial y} \end{matrix} \right]. \end{equation*}
    of \(f\text{.}\) This gives two formulas (one for each matrix entry of the derivative of the composition
    \begin{align*} \frac{\partial f}{\partial u} \amp = \frac{\partial f}{\partial x} \frac{\partial x}{\partial u} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial u}, \\ \frac{\partial f}{\partial v} \amp = \frac{\partial f}{\partial x} \frac{\partial x}{\partial v} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial v} . \end{align*}

    Example 1.3.6. Chain rule computation II.

    One often wants to consider a function \(f(x,y)\) on the plane as a function of the polar coordinates \(f(r, \theta)\text{.}\) This in fact is a composition. Indeed, when we write \(f(r, \theta)\) what we mean is the composition \(f \circ \mb{G}\) where \(\mb{G} (r , \theta) = (r \cos \theta, r \sin \theta)\) changes from cartesian to polar coordinates. I.e.
    \begin{equation*} f(r, \theta ) : = f(r \cos \theta , r \sin \theta) = f(x (r, \theta), y ( r, \theta)) f(r \cos \theta , r \sin \theta) = f \circ \mb{G}. \end{equation*}
    In Exercise 1.2.5.9, you computed the Jacobian matrix of \(\mb{G} (r , \theta) = (r \cos \theta, r \sin \theta)\) and should have found that
    \begin{equation*} \tder{}{\mb{G}} = \left[ \begin{matrix} \cos \theta \amp - r \sin \theta \\ \sin \theta \amp r \cos \theta \end{matrix} \right]. \end{equation*}
    Using chain rule, this then gives us that
    \begin{align*} \frac{\partial f}{\partial r} \amp = \frac{\partial f}{\partial x} \cos \theta + \frac{\partial f}{\partial y}\sin \theta , \\ \frac{\partial f}{\partial \theta} \amp = - \frac{\partial f}{\partial x} r \sin \theta + \frac{\partial f}{\partial y}r \cos \theta . \end{align*}

    Subsection 1.3.2 Tangent spaces

    To define the tangent space of a vector space at a point, we first consider the following result concerning directional derivatives.

    Corollary 1.3.7.

    If \(U\) is an open neighborhood of \(\mb{a}\) in \(V\) and \(f : U \to \mathbb{R}\) is differentiable at \(\mb{a}\) then for any vector \(\mb{v}\) in \(V\text{,}\)
    \begin{equation*} D_{\mb{v}} f (\mb{a} ) = \tder{\mb{a}}{f} (\mb{v} ). \end{equation*}
    In particular, for \(\mb{v}_1, \mb{v}_2\)
    \begin{equation*} D_{\mb{v}_1 + \mb{v}_2} f ( \mb{a} ) = D_{\mb{v}_1} f (\mb{a}) + D_{\mb{v}_2} f (\mb{a} ). \end{equation*}

    Proof.

    Let us reconsider the definition of the directional derivative
    \begin{equation*} D_\mb{v} f ( \mb{a}) = \left. \frac{d}{dt} f (\mb{a} + t\mb{v} )\right|_{t = 0}. \end{equation*}
    The right hand side is really the usual derivative at \(0\) of the composition \(f \circ \ell_{\mb{v}}\) where \(\ell_{\mb{v}} = \mb{a} + t \mb{v}\) is the function from \(\mathbb{R}\) to \(V\) parameterizing a line through \(\mb{a}\) of constant velocity \(\mb{v}\text{.}\) Using the chain rule then gives
    \begin{align*} D_\mb{v} f ( \mb{a}) \amp = \left. \frac{d}{dt} f (\mb{a} + t\mb{v} )\right|_{t = 0}, \\ \amp = \tder{0}{\left( f \circ \ell_{v} \right)} (1), \\ \amp = \tder{\mb{a}}{f} \circ \tder{0}{\ell_v} (1), \\ \amp = \tder{\mb{a}}{f} (\mb{v} ). \end{align*}
    But \(\tder{\mb{a}}{f}\) is a linear transformation so the second statement follows.
    Now we proceed to define the tangent space of a point \(\mb{a}\) in a real inner product space \(V\text{.}\) If it helps, you may think of \(V\) as \(\mathbb{R}^n\) or even as \(\mathbb{R}^2\) to build intuition. Recall that \(C^1 (U)\) is the vector space of all differentiable (real scalar valued) functions on a set \(U\text{.}\)

    Definition 1.3.8.

    A tangent vector \(\partial\) to \(\mb{a} \in V\) is a directional derivative operator
    \begin{equation*} \partial : C^1 (U) \to \mathbb{R} \end{equation*}
    which is defined by \(\partial f = D_\mb{v} f ( \mb{a})\) for some vector \(\mb{v}\) in \(V\text{.}\) Here \(U\) is any open set containing \(\mb{a}\text{.}\) The vector space of these operators is denoted \(T_{\mb{a}} V\) and called the tangent space of \(V\) at \(\mb{a}\text{.}\)
    When \(V = \mathbb{R}^m\text{,}\) we take \(\partial_1, \partial_2, \ldots, \partial_m\) for the tangent vectors \(D_{\mb{e}_1}, \ldots, D_{\mb{e}_m}\text{.}\) This forms a basis for \(T_{\mb{a}} \mathbb{R}^m\) as Proposition 1.3.9 implies.
    This definition may look rather strange to a student seeing it for the first time, after all, why not just say that \(\partial\) is \(\mb{v}\) and make the tangent space \(V\text{?}\) Indeed, we have the following simple result.

    Proposition 1.3.9.

    For any \(\mb{a}\) in an inner product space \(V\text{,}\) there is a linear isomorphism
    \begin{equation*} T : V \to T_{\mb{a}} V. \end{equation*}

    Proof.

    Define \(T\) by taking \(T (\mb{v})\) to \(D_{\mb{v}}\text{.}\) By Corollary 1.3.7 this is a linear transformation and by the definition of \(T_{\mb{a}} V\) it is an isomorphism.
    While \(V\) and \(T_{\mb{a}} V\) are linearly isomorphic, there are many reasons to make \(T_{\mb{a}} V\) its own vector space. Perhaps the most important one is that it replaces the ambient space with a new (linearly isomorphic) space `centered’ at \(\mb{a}\text{.}\) This gives every tangent vector its own identity and its own space in which to perform arithmetic and operations, thereby divining the spatial structure at \(\mb{a}\) from some other point. Indeed, there is a bit more to the sophisticated definition in that it should be independent of which open set \(U\) we choose around \(\mb{a}\text{.}\) Point being, tangent vectors of \(\mb{a}\) only care about what is happening very close to the point \(\mb{a}\text{.}\)
    Alternative viewpoints to the tangent space help in understanding it. One way is to see it as the space of differentiable paths through \(\mb{a}\) where two paths are identified if their tangent vectors are the same. The isomorphism can then be given by taking a path \(\gamma\) to \(\partial\) where \(\partial f = d_0 (f \circ \gamma ) (1)\text{.}\)
    Let us now recast a few definitions.

    Definition 1.3.10.

    A vector field is a function \(\mb{F}\) which takes a point \(\mb{a}\) and defines a vector \(\mb{F} (\mb{a})\) in \(T_{\mb{a}} V\text{.}\)

    Definition 1.3.11.

    If \(\mb{F} : U \to V_2\) is differentiable at \(\mb{a}\) in \(V_1\) with \(\mb{F} (\mb{a} ) = \mb{b}\text{,}\) the derivative of \(\mb{F}\) at \(\mb{a}\) is the linear transformation
    \begin{equation*} \tder{a}{\mb{F}} : T_\mb{a} V_1 \to T_{\mb{b}} V_2 \end{equation*}
    which is defined as
    \begin{equation*} \left[ \tder{a}{\mb{F}} ( \partial ) \right] (f) = \partial \left( f \circ \mb{F} \right). \end{equation*}
    These new definitions can replace our previous ones, or can live (slightly uncomfortably) along side them. However, in the latter case we will find our old viewpoints to be less conceptually workable as we advance our understanding. We may also use the new definition of the derivative as a linear transformation of tangent spaces to define tangent spaces of parameterized subspaces.

    Definition 1.3.12.

    Suppose \(V_1, V_2\) are finite dimensional real inner product spaces, \(U \subset V_1\) is an open subset and \(\mb{G} : U \to V_2\) is a differentiable, one to one function with \(\mb{G} (p) = q\text{.}\)
    1. The tangent space \(T_q X\) of \(X\) at q is the linear subspace
      \begin{equation*} T_q X = \text{im} \left( \tder{p}{\mb{G}}\right) \subset T_q V_2 \end{equation*}
    2. The embedded tangent space of \(X\) at q is the subspace
      \begin{equation*} \left\{ q + \mb{v} : D_{\mb{v}} \in T_q X \right\} \subset V_2 \end{equation*}
    To digest this last definition, consider the case of the graph \(X = \text{gr} (f)\) of a one variable function \(\mb{f}(t) = \left( t, f (t) \right)\text{.}\) As mentioned above, since \(\mb{f} : \mathbb{R} \to \mathbb{R}^2\) is a path, the derivative of \(\mb{f}\) at \(a\) is
    \begin{equation*} \tder{a}{\mb{f}} = \twovec{1}{f^\prime (a)} \end{equation*}
    To write this as a linear transformation of tangent spaces, give each tangent space a basis. At \(a\) we have that \(\partial_t\) is a basis of the one dimensional \(T_a \mathbb{R}\) and at \(\mb{a} = (a, f(a))\) we have the basis \(\left\{ \partial_x , \partial_y \right\}\text{.}\) Then the expression above is the matrix representing the linear transformation so that
    \begin{equation*} \tder{a}{\mb{f}} (\partial_t) = \partial_x + f^\prime (a) \partial_y \end{equation*}
    The image of this linear transformation then is all multiples of the vector \(\partial_x + f^\prime (a) \partial_y\) so that
    \begin{equation*} T_\mb{a} X = \{ \lambda \partial_x + \lambda f^\prime (a) \partial_y : \lambda \in \mathbb{R} \}. \end{equation*}
    or as column vectors
    \begin{equation*} T_\mb{a} X = \left\{ \twovec{\lambda}{\lambda f^\prime (a)} : \lambda \in \mathbb{R} \right\}. \end{equation*}
    Note that this is a linear subspace of \(T_{\mb{a}} \mathbb{R}^2\text{.}\) The embedded tangent space is not typically a linear subspace and is a subspace of the codomain \(\mathbb{R}^2\) of the function, not \(T_{\mb{a}} \mathbb{R}^2\text{.}\) This is just the line
    \begin{equation*} \left\{ \twovec{a}{f(a)} + \twovec{\lambda}{\lambda f^\prime (a)} : \lambda \in \mathbb{R}\right\} \end{equation*}
    or in Cartesian coordinates
    \begin{equation*} \left\{ \left(a + \lambda, f(a) + \lambda f^\prime (a) \right): \lambda \in \mathbb{R} \right\}. \end{equation*}
    Of course, this is just the usual tangent line to the graph of the function parameterized by \(\lambda\) and can be seen to be the set of points satisfying the usual familiar implicit equation
    \begin{equation*} y = f(a) + f^\prime (a) (x - a). \end{equation*}

    Subsection 1.3.3 Level sets and their tangent spaces

    When working with a set of variables, one often has additional constraints that come into play. For example, one might have some function \(f (E, m , v)\) on three variables, \(E, m, v\) where \(E\) is the kinetic energy of a particle, \(m\) is its mass and \(v\) its velocity. As we all learn in middle school physics, these three variables satisfy an equation
    \begin{equation*} E = \frac{1}{2} m v^2 . \end{equation*}
    So our function is really only interesting on the points \((E, m, v) \in \mathbb{R}^3\) satisfying this equation (or constraint). As it turns out, this constraint defines a surface in three dimensional space and we may wish to find its two dimensional tangent space at a given point. To do this, we consider the function
    \begin{equation*} g (E, m, v) = E - \frac{1}{2} m v^2 . \end{equation*}
    Recall that the zero level set \(X\) (or level surface in this case) of \(g\) is the subspace satisfying \(g (E, m, v) = 0\text{.}\) If the point \(p = (E_0, m_0, v_0)\) lies on \(X\text{,}\) we wish to find the tangent vectors in \(T_p \mathbb{R}^3\) which are tangent to \(X\text{.}\) To do this, we consider any given differentiable path \(\gamma: (-1, 1) \to X\) which passes through \(p\) at \(0\text{.}\) The tangent (or velocity) vector to such paths at \(0\) should give all of the tangent vectors to \(X\text{.}\) But if we apply the chain rule to this argument, then we can see that all such tangent vectors satisfy the condition
    \begin{equation} \tder{p}{g} (\mb{v} ) = 0.\tag{1.3.1} \end{equation}
    For our running example, we can compute for \(\mb{v} = \tangthree{a}{b}{c}\) the equation yields
    \begin{equation*} 0 = \tder{p}{g} (\mb{v} ) = \left[ \begin{matrix} 1 \amp - \frac{1}{2}v_0^2 \amp - m_0 v_0 \end{matrix} \right] \tangthree{a}{b}{c} = a - \frac{1}{2}v_0^2 b - m_0 v_0 c . \end{equation*}
    which is an equation for a plane in the three dimensional \(T_p \mathbb{R}^3\text{.}\)
    In fact, it is a result of the Implicit Function Theorem that, if \(\tder{p}{g}\) is not zero, then this equation defines the tangent space to \(X\) at \(p\text{.}\) Let us take this as an unproven theorem.

    Theorem 1.3.13. Implicit Function Theorem Consequence.

    Let \(U\) be an open set in \(\mathbb{R}^m\text{,}\) \(\mathbb{G} : U \to \mathbb{R}^n\) and \(X\) the level set of all points \(p\) with \(\mb{G} (p) = q\) for some fixed \(q\text{.}\) If \(\mb{G}\) is differentiable at \(p\) and \(\rk (\tder{p}{\mb{G}} ) = n\) then the tangent space to \(X\) at \(p\) as a subspace of \(T_p \mathbb{R}^m\) is
    \begin{equation*} T_p X = \left\{ \partial \in T_p \mathbb{R}^m : \tder{p}{\mb{G}} (\partial ) = 0 \right\} . \end{equation*}
    This theorem may appear too abstract, so let us do a computation to show its simplicity.

    Example 1.3.14. Tangent space of a quadric.

    Consider the quadric \(X\) defined by the equation
    \begin{equation*} x^2 + y^2 - z^2 = 1 \end{equation*}
    and take a point \(p = (1, 1, 1)\) on \(X\text{.}\) The subspace \(X\) is a surface in three dimensions, but it is not the graph of a single function (meaning it is not the set of points satisfying \(z = f(x,y)\) for some function \(f(x,y)\)), so our previous method for finding the tangent plane to \(X\) cannot be used. Nevertheless, we can see that the constraint can be written as
    \begin{equation*} g(x,y, z) = 1 \end{equation*}
    where \(g(x,y, z) = x^2 + y^2 - z^2\text{.}\) The derivative of \(g\) is the matrix
    \begin{equation*} \tder{}{g} = \left[ \begin{matrix} 2x \amp 2y \amp -2z \end{matrix} \right] \end{equation*}
    and at our point \(p\) this gives
    \begin{equation*} \tder{p}{g} = \left[ \begin{matrix} 2 \amp 2 \amp -2 \end{matrix} \right]. \end{equation*}
    So the defining equation for the tangent space \(T_p X\) is simply
    \begin{equation*} 2 a + 2 b - 2c = 0 \end{equation*}
    or
    \begin{equation*} a + b - c = 0 \end{equation*}
    where the vector \(\tangthree{a}{b}{c}\) represents \(a \partial_1 + b \partial_2 + c \partial_3\text{.}\) Note that if we want to view this plane as a plane in \(\mathbb{R}^3\) and not simply \(T_p \mathbb{R}^3\text{,}\) we have to translate the origin to \(p\) so that \(a = (x - 1)\text{,}\) \(b = (y - 1)\) and \(c = (z - 1)\text{.}\) This then gives the tangent plane to \(X\) at \(p\) as a subspace of \(\mathbb{R}^3\) as
    \begin{equation*} (x - 1) + (y - 1) - (z - 1) = 0 \end{equation*}
    or
    \begin{equation*} x + y - z = 1. \end{equation*}
