Proposition 1.5.4.

Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) is scalar function. If \(f\) achieves a local minimum or maximum at \(p\text{,}\) then \(p\) is a critical point of \(f\text{.}\)

Proof.

If \(f\) is not differentiable at \(p\text{,}\) then \(p\) is a critical point and we need not argue further. Otherwise, suppose \(f\) is differentiable at \(p\) and it is a local minimum (the argument for maximum is similar). If \(p\) is not a critical point then \(\nabla_p f \ne \mb{0}\) and, by PropositionÂ 1.4.13, there is a path \(\gamma : (-\varepsilon , \varepsilon) \to U\) for which \(\gamma (0) = p\) and \(f \circ \gamma\) is strictly increasing. But then for any \(t\) with \(-\varepsilon \lt t \lt 0\) we have \(f ( \gamma (t)) \lt f (p)\text{.}\) This contradicts the assumption that \(p\) is a local minimum for \(f\text{.}\)
