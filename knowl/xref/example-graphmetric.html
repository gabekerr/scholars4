Example 1.4.2. Metrics of graphs.

One class of examples of parameterized surfaces comes from considering graphs of functions. For example, if \(f(x,y)\) is a differentiable function on the plane \(\mathbb{R}^2\) then we can take its graph
\begin{equation*} X = \{ (x, y, z) : z = f(x,y) \}. \end{equation*}
Such a subspace is, in a sense, already parameterized as we can define \(\mb{G} : \mathbb{R}^2 \to \mathbb{R}^3\) via
\begin{equation*} \mb{G} (x, y) = (x, y, f(x,y)). \end{equation*}
Taking the derivative gives
\begin{equation*} \tder{}{\mb{G}} = \left[ \begin{matrix} 1 \amp 0 \\ 0 \amp 1 \\ \frac{\partial f}{\partial x} \amp \frac{\partial f}{ \partial y} \end{matrix} \right]. \end{equation*}
Thus if we look at the tangent vectors
\begin{align*} \tder{}{\mb{G}} ( \partial_x ) \amp = \partial_x + \frac{\partial f}{\partial x} \partial_z \\ \tder{}{\mb{G}} ( \partial_y) \amp = \partial_y + \frac{\partial f}{\partial y} \partial_z \end{align*}
we see that their dot product is
\begin{equation*} \tder{}{\mb{G}} ( \partial_x ) \cdot \tder{}{\mb{G}} ( \partial_y) = \frac{\partial f}{\partial x} \frac{\partial f}{\partial y} . \end{equation*}
This is generally non-zero, even though \(\partial_x \cdot \partial_y = 0\text{.}\) This is again because the image tangent vectors fail to be orthogonal.
