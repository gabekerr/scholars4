Corollary 1.3.7.

If \(U\) is an open neighborhood of \(\mb{a}\) in \(V\) and \(f : U \to \mathbb{R}\) is differentiable at \(\mb{a}\) then for any vector \(\mb{v}\) in \(V\text{,}\)
\begin{equation*} D_{\mb{v}} f (\mb{a} ) = \tder{\mb{a}}{f} (\mb{v} ). \end{equation*}
In particular, for \(\mb{v}_1, \mb{v}_2\)
\begin{equation*} D_{\mb{v}_1 + \mb{v}_2} f ( \mb{a} ) = D_{\mb{v}_1} f (\mb{a}) + D_{\mb{v}_2} f (\mb{a} ). \end{equation*}

Proof.

Let us reconsider the definition of the directional derivative
\begin{equation*} D_\mb{v} f ( \mb{a}) = \left. \frac{d}{dt} f (\mb{a} + t\mb{v} )\right|_{t = 0}. \end{equation*}
The right hand side is really the usual derivative at \(0\) of the composition \(f \circ \ell_{\mb{v}}\) where \(\ell_{\mb{v}} = \mb{a} + t \mb{v}\) is the function from \(\mathbb{R}\) to \(V\) parameterizing a line through \(\mb{a}\) of constant velocity \(\mb{v}\text{.}\) Using the chain rule then gives
\begin{align*} D_\mb{v} f ( \mb{a}) \amp = \left. \frac{d}{dt} f (\mb{a} + t\mb{v} )\right|_{t = 0}, \\ \amp = \tder{0}{\left( f \circ \ell_{v} \right)} (1), \\ \amp = \tder{\mb{a}}{f} \circ \tder{0}{\ell_v} (1), \\ \amp = \tder{\mb{a}}{f} (\mb{v} ). \end{align*}
But \(\tder{\mb{a}}{f}\) is a linear transformation so the second statement follows.
