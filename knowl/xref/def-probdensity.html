Definition 3.3.5.

Given a region \(D\) in \(\mathbb{R}^n\text{,}\) a function \(\rho : D \to \mathbb{R}\) is a probability density if
  1. \(\rho\) takes only non-negative values,
  2. \(\rho\) is integrable on \(D\) and
    \begin{equation*} \idotsint_D \rho (x_1, \ldots, x_n) \, \tder{}{V} = 1. \end{equation*}
