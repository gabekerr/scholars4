Proposition 4.1.7.

Suppose \(f : \mathbb{R}^n \to \mathbb{R}\) is a smooth scalar function on Euclidean space for which \(c\) is not a critical value. Then the smooth level subspace
\begin{equation*} X = \left\{ \mb{x} \in \mathbb{R}^n : f (\mb{x} ) = c \right\} \end{equation*}
is orientable with orientation form
\begin{equation*} \orient^X = \star \tder{}{f} \end{equation*}

Proof.
