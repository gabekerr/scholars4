Proposition 1.1.3.

Suppose \(U\) is an open set in \(V\text{,}\) \(\lambda\) a scalar in \(K\text{,}\) \(h: K \to K\) is a continuous function at \(L\text{,}\) and \(f, g : U \to K\) are scalar functions such that
\begin{equation*} \lim_{\mb{x} \to \mb{c}} f(\mb{x} ) = L, \hspace{.1in} \lim_{\mb{x} \to \mb{c}} g(\mb{x} ) = M . \end{equation*}
Then
  1. \begin{equation*} \lim_{\mb{x} \to \mb{c}} (\lambda f + g) (\mb{x} ) = \lambda L + M \text{,} \end{equation*}
  2. \begin{equation*} \lim_{\mb{x} \to \mb{c}} f (\mb{x} ) \, g(\mb{x}) = L M , \end{equation*}
  3. if \(M \ne 0\text{,}\)
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} \frac{f (\mb{x} )}{g (\mb{x} )} = \frac{L}{M}. \end{equation*}
  4. \begin{equation*} \lim_{\mb{x} \to \mb{c}} (h \circ f)(\mb{x} ) = h (L). \end{equation*}
