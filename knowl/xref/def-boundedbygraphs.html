Definition 2.2.4.

A compact domain \(D\) in \(\mathbb{R}^2\) is bounded by continuous graphs of \(x\) if there is an interval \([a,b]\) and continuous functions \(g_1, g_2 : [a,b] \to \mathbb{R}\) for which
\begin{equation*} D = \left\{ (x,y) : a \leq x \leq b , g_1 (x) \leq y \leq g_2 (x) \right\}. \end{equation*}
The domain \(D\) is \(\textbf{bounded by continuous graphs of } y\) if there is an interval \([c,d]\) and continuous functions \(h_1, h_2 : [c,d] \to \mathbb{R}\) for which
\begin{equation*} D = \left\{ (x,y) : c \leq y \leq d , h_1 (y) \leq x \leq h_2 (y) \right\}. \end{equation*}
