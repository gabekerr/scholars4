Lemma 1.5.8.

Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\text{,}\) \(f: U \to \mathbb{R}\) is scalar function with continuous second partial derivatives and let \(\gamma : (-\varepsilon, \varepsilon) \to U\) be a differentiable path. Assume \(\gamma(0) = p\) is a critical point of \(f\text{.}\) Then
\begin{equation*} \left. \frac{d^2}{dt^2} (f \circ \gamma)\right|_{t = 0} = \gamma^\prime (0)^T \textnormal{Hess}_{\gamma (0)} (f) \gamma^\prime (0). \end{equation*}

Proof.

This is an exercise in using the chain rule (twice). The first use shows
\begin{equation*} \frac{d}{dt} (f \circ \gamma) = \tder{\gamma (t)}{f} \cdot \gamma^\prime (t). \end{equation*}
Here we have a matrix product of the Jacobian matrix \(\tder{\gamma(t)}{f}\) with the tangent vector \(\gamma^\prime (t)\text{.}\) Taking the derivative again with respect to \(t\) forces us to use the product rule
\begin{align*} \frac{d^2}{dt^2} (f \circ \gamma) \amp = \frac{d}{dt} \left( \tder{\gamma (t)}{f} \cdot \gamma^\prime (t) \right), \\ \amp =\left( \frac{d}{dt} \tder{\gamma (t)}{f} \right) \gamma^\prime (t) + \tder{\gamma (t)}{f} \left( \frac{d}{dt} \gamma^\prime (t) \right). \end{align*}
Now, examining the second summand and evaluating at \(t = 0\) will give
\begin{equation*} \tder{\gamma (0)}{f} \left( \left. \frac{d}{dt} \gamma^\prime (t) \right|_{t = 0} \right) = \tder{p}{f} \left( \left. \frac{d}{dt} \gamma^\prime (t) \right|_{t = 0} \right) = 0 \end{equation*}
because \(p\) is a critical point of \(f\text{.}\) So we focus on the first summand and, in particular, the derivative
\begin{equation*} \frac{d}{dt} \tder{\gamma (t)}{f}. \end{equation*}
The function \(\tder{\gamma (t)}{f}\) can be written as the row vector
\begin{equation*} \left[ \begin{matrix} \left( \frac{\partial f}{\partial x_1} \circ \gamma \right) (t) \amp \cdots \amp \left( \frac{\partial f}{\partial x_m} \circ \gamma \right) (t) \end{matrix} \right]. \end{equation*}
Letting \(\gamma (t) = (\gamma_1 (t) , \ldots , \gamma_m (t))\) and using the chain rule on the \(j\)-th entry above gives
\begin{equation*} \frac{d}{dt}\left( \frac{\partial f}{\partial x_j} \circ \gamma \right) (t) = \frac{\partial^2 f}{\partial x_1 \partial x_j} \gamma^\prime_1 (t) + \frac{\partial^2 f}{\partial x_2 \partial x_j} \gamma^\prime_2(t) + \cdots + \frac{\partial^2 f}{\partial x_m \partial x_j} \gamma^\prime_m (t). \end{equation*}
But this is simply the \(j\)-th entry of the matrix product
\begin{equation*} \gamma^\prime (t)^T \cdot \textnormal{Hess}_{\gamma (t)}(f). \end{equation*}
Thus we obtain the formula
\begin{equation*} \frac{d}{dt} \tder{\gamma (t)}{f} = \gamma^\prime (t)^T \cdot \textnormal{Hess}_{\gamma (t)} (f) \end{equation*}
and conclude that, at \(t = 0\)
\begin{equation*} \left. \frac{d^2}{dt^2} (f \circ \gamma)\right|_0 = \gamma^\prime (0)^T \textnormal{Hess}_{\gamma (0)}(f) \gamma^\prime (0). \end{equation*}
