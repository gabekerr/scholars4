Theorem 1.3.13. Implicit Function Theorem Consequence.

Let \(U\) be an open set in \(\mathbb{R}^m\text{,}\) \(\mathbb{G} : U \to \mathbb{R}^n\) and \(X\) the level set of all points \(p\) with \(\mb{G} (p) = q\) for some fixed \(q\text{.}\) If \(\mb{G}\) is differentiable at \(p\) and \(\rk (\tder{p}{\mb{G}} ) = n\) then the tangent space to \(X\) at \(p\) as a subspace of \(T_p \mathbb{R}^m\) is
\begin{equation*} T_p X = \left\{ \partial \in T_p \mathbb{R}^m : \tder{p}{\mb{G}} (\partial ) = 0 \right\} . \end{equation*}
