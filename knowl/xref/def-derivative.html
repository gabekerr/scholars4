Definition 1.2.1.

Let \(V_1\) and \(V_2\) be inner product spaces and suppose \(\mb{F} : U \to V_2\) is a function defined in a neighborhood \(\mb{F}\) of \(\mb{a}\text{.}\) The derivative \(\tder{\mb{a}}{\mb{F}}\) of \(\mb{F}\) at \(\mb{a}\) is a linear transformation from \(V_1\) to \(V_2\) for which
\begin{equation} \lim_{\mb{x} \to \mb{a}} \frac{\| \mb{F} ( \mb{x} ) - \mb{F} (\mb{a} ) - \tder{\mb{a}}{\mb{F}} (\mb{x} - \mb{a}) \|}{\| \mb{x} - \mb{a} \|} = 0.\tag{1.2.1} \end{equation}
If \(\tder{\mb{a}}{\mb{F}}\) exists we say that \(\mb{F}\) is differentiable at \(\mb{a}\).
