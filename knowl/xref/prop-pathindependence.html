Proposition 5.3.9. Path Independence.

Let \(\mb{F}\) be a vector field defined on an open set \(U\) of \(\mathbb{R}^n\text{.}\) Then \(\mb{F}\) is conservative if and only if the vector line integral
\begin{equation*} \int_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}} \end{equation*}
on a smooth connected oriented curve in \(U\) depends only on its initial and terminal points.

Proof.

If one assumes \(\mb{F}\) is conservative, then the fundamental theorem of line integrals immediately shows it is path independent. On the other hand, if it is path independent, for each connected component \(W\) of \(U\text{,}\) we choose a point \(\mb{a} \in W\) and define the function
\begin{equation} V (\mb{x} ) = \int_{P (\mb{x} )} \mb{F} \cdot \tder{}{\mb{r}}\tag{5.3.1} \end{equation}
where \(P ( \mb{x})\) is any path from \(\mb{a}\) to \(\mb{x}\text{.}\) Showing \(V\) is indeed a potential for \(\mb{F}\) is left as an exercise.
