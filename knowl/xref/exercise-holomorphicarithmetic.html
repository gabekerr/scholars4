Exercise 2.

Show the following results
(a)
Verify that a constant vector field is solenoidal and satisfies the cross-partials test.
(b)
Verify that \(\mb{F} = x \partial_x - y \partial_y\) is solenoidal and satisfies the cross-partials test.
(c)
If \(\mb{F}_1 = P_1 \partial_x + Q_1 \partial_y\) and \(\mb{F}_2 = P_2 \partial_x + Q_2 \partial_y\) are both solenoidal and satisfy the cross partial tests, then so does \(\mb{G} = (P_1 P_2 - Q_1 Q_2) \partial_x + (P_1 Q_2 + P_2 Q_1) \partial_y\text{.}\)
(d)
If \(\mb{F} = P \partial_x + Q \partial_y\) is solenoidal and satisfy the cross partial tests, then so does
\begin{equation*} \mb{F}_{inv} = \frac{1}{P^2 + Q^2} \left( P \partial_x - Q \partial_y \right). \end{equation*}
