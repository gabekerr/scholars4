    Section 1.5 Critical points

    Looking over the propositions in the last section you may notice that the condition `assume the gradient does not vanish’ appears multiple times. So what happens if this fails? Well, that’s where things get interesting...

    Definition 1.5.1.

    Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) is scalar function. We call \(p \in U\) a critical point of \(f\) if either \(f\) is not differentiable at \(p\) or if
    \begin{equation*} \nabla_p f = \mb{0}. \end{equation*}
    The value \(f(p)\) is called a \(\textbf{critical value}\) of \(f\text{.}\)
    One can be more general than this and say that \(p\) is a critical point of a vector valued \(\mb{F} : U \to \mathbb{R}^n\) if \(\tder{p}{\mb{F}}\) is not of full rank, but we will stay with scalar functions for the time being.

    Example 1.5.2. Computing critical points.

    Let \(f(x,y) = y \cos (x)\) and compute the gradient to find
    \begin{equation*} \nabla f = \langle - y \sin x , \cos x \rangle . \end{equation*}
    The critical points are points \((x,y)\) where \(\nabla f = \langle 0, 0 \rangle\) so that both equations
    \begin{align*} -y \sin x \amp = 0,\\ \cos x \amp = 0 \end{align*}
    are satisfied. The latter equation is satisfied only when \(x = \frac{\pi}{2} + \pi k\) for some integer \(k\text{.}\) For such \(x\text{,}\) \(\sin x = \pm 1\) so the only way the first equation is satisfied for these values of \(x\) is if \(y = 0\text{.}\) Thus the critical points of the function \(f\) is the set
    \begin{equation*} \left\{ \left( \frac{\pi}{2} + k , 0 \right) : k \text{ is an integer} \right\}. \end{equation*}
    Since the \(y\) coordinate of any critical point is \(0\text{,}\) the only critical value is \(0\text{.}\)

    Subsection 1.5.1 Local extrema

    The derivative of a function gives local information about that function. Let us make a definition of one such piece of information of a scalar function.

    Definition 1.5.3.

    Suppose \(X\) is a subset of \(\mathbb{R}^m\text{,}\) \(p\) a point of \(X\) and \(f : X \to \mathbb{R}\) is a scalar function on \(X\text{.}\) We say that \(f\) achieves a local minimum (respectively local maximum) at \(p\) if there is a neighborhood \(U\) of \(p\) in \(X\) such that \(f(p) \leq f(\tilde{p})\) (respectively \(f(p) \geq f(\tilde{p})\)) for all \(\tilde{p}\) of \(U\text{.}\)
    Local minima and maxima are extremely important notions in applications as they often describe points where systems are stable or optimized. Finding them can sometimes be a challenge, but one very helpful tool we can use is the following proposition.

    Proposition 1.5.4.

    Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) is scalar function. If \(f\) achieves a local minimum or maximum at \(p\text{,}\) then \(p\) is a critical point of \(f\text{.}\)

    Proof.

    If \(f\) is not differentiable at \(p\text{,}\) then \(p\) is a critical point and we need not argue further. Otherwise, suppose \(f\) is differentiable at \(p\) and it is a local minimum (the argument for maximum is similar). If \(p\) is not a critical point then \(\nabla_p f \ne \mb{0}\) and, by Proposition 1.4.13, there is a path \(\gamma : (-\varepsilon , \varepsilon) \to U\) for which \(\gamma (0) = p\) and \(f \circ \gamma\) is strictly increasing. But then for any \(t\) with \(-\varepsilon \lt t \lt 0\) we have \(f ( \gamma (t)) \lt f (p)\text{.}\) This contradicts the assumption that \(p\) is a local minimum for \(f\text{.}\)
    Let us be clear about how we use Proposition 1.5.4. If we want to find a local extreme point, we can look for critical points. However, it is not the case a given critical point will always be a local extreme point. Let us illustrate this with our three prototype scalar functions on \(\mathbb{R}^2\text{.}\)

    Example 1.5.5. Chain rule computation II.

    Let
    \begin{equation*} Q_{++} (x, y) = x^2 + y^2. \end{equation*}
    It is clear that \(Q_0 (x,y)\) is differentiable and
    \begin{equation*} \nabla Q_{++} = \langle 2x , 2y \rangle. \end{equation*}
    We also see that the only critical point of \(Q_0\) is the origin \((0, 0)\text{.}\) Thus the only local extreme point for \(Q_0\) is the origin, and as \(Q_0\) is always positive otherwise we have that \(Q_0\) achieves a local (and global) minimum at the origin.
    A similar argument shows that
    \begin{equation*} Q_{--} (x, y) = -x^2 - y^2 \end{equation*}
    has exactly one local (and global) maximum at the origin.
    Now consider the last prototype
    \begin{equation*} Q_{+-} (x, y) = x^2 - y^2 \end{equation*}
    and observe
    \begin{equation*} \nabla Q_{+-} = \langle 2x , -2y \rangle. \end{equation*}
    So here again \((0, 0)\) is the only critical point. However, we see that along the \(x\)-axis away from the origin, \(Q_{+-}\) is positive and along the \(y\)-axis away from the origin, \(Q_{+-}\) is negative. So there is no chance that \(Q_{+-}\) will have a local minimum or maximum at the origin. Critical points of this type have their own name as we now define.

    Definition 1.5.6.

    Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) is scalar function. If \(p\) is a critical point that is neither a local minimum nor a local maximum, then \(p\) is called a saddle point of \(f\text{.}\)
    In general, saddle points can be quite complicated. Perhaps surprisingly though we can fully describe the local behavior of a function near a differentiable critical point so long as we have something called non-degeneracy. To get to this notion, we first define the Hessian.

    Definition 1.5.7.

    Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) is scalar function with continuous second partial derivatives. The Hessian of \(f\) is the matrix valued function
    \begin{equation*} \textnormal{Hess} (f) = \left[ \begin{matrix} \frac{\partial^2 f}{\partial x_1^2} \amp \frac{\partial^2 f}{\partial x_1 \partial x_2} \amp \cdots \amp \frac{\partial^2 f}{\partial x_1 \partial x_m} \\ \frac{\partial^2 f}{\partial x_2 \partial x_1} \amp \frac{\partial^2 f}{\partial x_2^2} \amp \cdots \amp \frac{\partial^2 f}{\partial x_2 \partial x_m} \\ \vdots \amp \vdots \amp \amp \vdots \\ \frac{\partial^2 f}{\partial x_m \partial x_1} \amp \frac{\partial^2 f}{\partial x_m \partial x_2} \amp \cdots \amp \frac{\partial^2 f}{\partial x_m^2} \end{matrix} \right] \end{equation*}
    Write \(\text{Hess}_p (f)\) for the Hessian of \(f\) evaluated at \(p\text{.}\) We say a critical point \(p\) of \(f\) is non-degenerate if \(\text{Hess}_p (f)\) has non-zero determinant.

    Subsection 1.5.2 Second derivative test

    Note that by Clairaut’s Theorem, the Hessian of \(f\) is a symmetric \(m \times m\) matrix and thus can be linked to a quadratic form. In fact, the Hessian can be thought of as a second derivative for scalar functions. The next lemma shows how useful this can be.

    Lemma 1.5.8.

    Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\text{,}\) \(f: U \to \mathbb{R}\) is scalar function with continuous second partial derivatives and let \(\gamma : (-\varepsilon, \varepsilon) \to U\) be a differentiable path. Assume \(\gamma(0) = p\) is a critical point of \(f\text{.}\) Then
    \begin{equation*} \left. \frac{d^2}{dt^2} (f \circ \gamma)\right|_{t = 0} = \gamma^\prime (0)^T \textnormal{Hess}_{\gamma (0)} (f) \gamma^\prime (0). \end{equation*}

    Proof.

    This is an exercise in using the chain rule (twice). The first use shows
    \begin{equation*} \frac{d}{dt} (f \circ \gamma) = \tder{\gamma (t)}{f} \cdot \gamma^\prime (t). \end{equation*}
    Here we have a matrix product of the Jacobian matrix \(\tder{\gamma(t)}{f}\) with the tangent vector \(\gamma^\prime (t)\text{.}\) Taking the derivative again with respect to \(t\) forces us to use the product rule
    \begin{align*} \frac{d^2}{dt^2} (f \circ \gamma) \amp = \frac{d}{dt} \left( \tder{\gamma (t)}{f} \cdot \gamma^\prime (t) \right), \\ \amp =\left( \frac{d}{dt} \tder{\gamma (t)}{f} \right) \gamma^\prime (t) + \tder{\gamma (t)}{f} \left( \frac{d}{dt} \gamma^\prime (t) \right). \end{align*}
    Now, examining the second summand and evaluating at \(t = 0\) will give
    \begin{equation*} \tder{\gamma (0)}{f} \left( \left. \frac{d}{dt} \gamma^\prime (t) \right|_{t = 0} \right) = \tder{p}{f} \left( \left. \frac{d}{dt} \gamma^\prime (t) \right|_{t = 0} \right) = 0 \end{equation*}
    because \(p\) is a critical point of \(f\text{.}\) So we focus on the first summand and, in particular, the derivative
    \begin{equation*} \frac{d}{dt} \tder{\gamma (t)}{f}. \end{equation*}
    The function \(\tder{\gamma (t)}{f}\) can be written as the row vector
    \begin{equation*} \left[ \begin{matrix} \left( \frac{\partial f}{\partial x_1} \circ \gamma \right) (t) \amp \cdots \amp \left( \frac{\partial f}{\partial x_m} \circ \gamma \right) (t) \end{matrix} \right]. \end{equation*}
    Letting \(\gamma (t) = (\gamma_1 (t) , \ldots , \gamma_m (t))\) and using the chain rule on the \(j\)-th entry above gives
    \begin{equation*} \frac{d}{dt}\left( \frac{\partial f}{\partial x_j} \circ \gamma \right) (t) = \frac{\partial^2 f}{\partial x_1 \partial x_j} \gamma^\prime_1 (t) + \frac{\partial^2 f}{\partial x_2 \partial x_j} \gamma^\prime_2(t) + \cdots + \frac{\partial^2 f}{\partial x_m \partial x_j} \gamma^\prime_m (t). \end{equation*}
    But this is simply the \(j\)-th entry of the matrix product
    \begin{equation*} \gamma^\prime (t)^T \cdot \textnormal{Hess}_{\gamma (t)}(f). \end{equation*}
    Thus we obtain the formula
    \begin{equation*} \frac{d}{dt} \tder{\gamma (t)}{f} = \gamma^\prime (t)^T \cdot \textnormal{Hess}_{\gamma (t)} (f) \end{equation*}
    and conclude that, at \(t = 0\)
    \begin{equation*} \left. \frac{d^2}{dt^2} (f \circ \gamma)\right|_0 = \gamma^\prime (0)^T \textnormal{Hess}_{\gamma (0)}(f) \gamma^\prime (0). \end{equation*}
    The following corollary follows from this lemma.

    Theorem 1.5.9.

    Suppose \(U\) is a domain with metric in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) is scalar function with continuous second partial derivatives and \(p\) is a non-degenerate critical point of \(f\text{.}\) By Sylvester’s Theorem, there are \(m_-\) negative eigenvalues of \(\textnormal{Hess}_p\) and \(m_+\) positive eigenvalues of \(\textnormal{Hess}_p\text{.}\) Suppose
    \begin{align*} \Gamma_- : \mathbb{R}^{m_-} \amp \to \mathbb{R}^m, \\ \Gamma_+ : \mathbb{R}^{m_+} \amp \to \mathbb{R}^m, \end{align*}
    are differentiable one-to-one maps sending \(\mb{0}\) to \(p\) and \(T_\mb{0} \mathbb{R}^{m_-}\) and \(T_\mb{0} \mathbb{R}^{m_+}\) to the negative and positive eigenspaces of \(\textnormal{Hess}_p\) respectively. Then \(f \circ \Gamma_-\) has a local maximum at \(\mb{0}\) and \(f \circ \Gamma_+\) has a local minimum at \(\mb{0}\text{.}\)

    Proof.

    We sketch the proof for \(f \circ \Gamma_-\text{,}\) but skip a basic estimate. Using the continuity of the second derivatives and the largest negative eigenvalue, we have that there exists a global \(\varepsilon\) such that any vector \(\mb{v}\) in \(\mathbb{R}^{m_-}\) with \(\| \mb{v} \| \lt \varepsilon\) the function \(f \circ \Gamma_-\) is concave on the line from \(- \mb{v}\) to \(\mb{v}\text{.}\) Indeed, taking the path \(\gamma : (-\varepsilon , \varepsilon ) \to \mathbb{R}^{m}\) defined by \(\gamma (t) = \Gamma_- (t \mb{v})\text{,}\) we have that
    \begin{equation*} \left. \frac{d}{dt} (f \circ \gamma ) (t) \right|_{t = 0} = \nabla_p f \cdot \gamma^\prime (0) = 0 \end{equation*}
    because \(p\) is critical and
    \begin{equation*} \left. \frac{d^2}{dt^2} (f \circ \gamma ) (t) \right|_{t = 0} = \gamma^\prime (0)^T \textnormal{Hess}_{\gamma (0)} (f) \gamma^\prime (0) = \mb{v}^T \textnormal{Hess}_{p} (f) \mb{v} \lt 0 \end{equation*}
    because \(\mb{v}\) is in the negative eigenspace of \(\textnormal{Hess}_p (f)\text{.}\) Using the one variable second derivative test, as well as the continuity of \(f\) and its second derivatives, we can obtain a global \(\varepsilon\) for which \(f(\Gamma_- (t\mb{v})) \lt f(p)\) implying \(p\) is (strict) local maximum. An analogous argument holds for the positive eigenspaces and the minimum.
    An important, but much less refined, corollary of this is the following result.

    Corollary 1.5.10. Second Derivative Test.

    Suppose \(U\) is a domain in \(\mathbb{R}^m\text{,}\) \(f: U \to \mathbb{R}\) is scalar function with continuous second partial derivatives and \(p\) is a non-degenerate critical point of \(f\text{.}\) Then
    1. \(f\) achieves a local minimum at \(p\) if and only if \(\textnormal{Hess}_p (f)\) is positive definite,
    2. \(f\) achieves a local maximum at \(p\) if and only if \(\textnormal{Hess}_p (f)\) is negative definite,
    3. \(f\) has a saddle point at \(p\) if \(\textnormal{Hess}_p (f)\) has both positive and negative eigenvalues.

    Subsection 1.5.3 The Morse Lemma

    In fact, much more can be said about the local behavior of functions near critical points. We will mention a few of these things, but omit proofs as many of them occur much later in an undergraduate (or graduate) mathematics education. Perhaps first among such statements is second order approximations.

    Theorem 1.5.11.

    Suppose \(U\) is a domain in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) has continuous second partial derivatives at \(\mb{a}\text{.}\) Let
    \begin{equation*} Q(\mb{x}) = f(\mb{a}) + \tder{\mb{a}}{f} \cdot (\mb{x} - \mb{a}) + \frac{1}{2} (\mb{x} - \mb{a})^T \textnormal{Hess}_\mb{a} (f) (\mb{x} - \mb{a}) \end{equation*}
    and call \(Q\) the second order approximation of \(f\text{.}\) If
    \begin{equation*} E (\mb{x}) = f(\mb{x}) - Q (\mb{x}) \end{equation*}
    is the error function, then
    \begin{equation*} \lim_{\mb{x} \to \mb{a}} \frac{E (\mb{x})}{\|\mb{x} - \mb{a}\|^2} = 0. \end{equation*}
    This theorem justifies the notion that \(\textnormal{Hess}(f)\) plays the role of a second derivative. Indeed, it is the second order term in a multivariable Taylor series expansion of our function \(f\) around \(\mb{a}\text{.}\) The function \(Q(\mb{x})\) is really the sum of our linear approximation and a quadratic form with associated matrix \(\textnormal{Hess}(f)\text{.}\) This is the analog to first approximating by a tangent line, and then by a parabola. The parabola of course gives much more information than the line, especially when it comes to understanding critical behavior.
    To drive home how precise this can be, we end this section with a relatively recent lemma (proven in 1929 by Marston Morse).

    Theorem 1.5.12. Morse Lemma.

    Suppose \(U\) is a domain in \(\mathbb{R}^m\) and \(f: U \to \mathbb{R}\) has continuous second partial derivatives. Let \(p\) be a non-degenerate critical point of \(f\) such that \(\textnormal{Hess}_p (f)\) has \(r\) positive eigenvalues and \(s\) negative eigenvalues. Then there is an open ball \(B\) centered at the origin in \(\mathbb{R}^m\) and a one-to-one differentiable function \(\Phi : B \to U\) with \(\Phi (\mb{0}) = p\) for which
    \begin{equation*} (f \circ \Phi) (x_1, \ldots, x_r, y_1, \ldots, y_s) = f(p) + x_1^2 + \cdots + x_r^2 - y_1^2 - \cdots - y_s^2. \end{equation*}
    The function \(\Phi\) should be thought of as giving a new coordinate system to a neighborhood around \(p\text{.}\) Then this theorem asserts that there is a coordinate system for which the quadratic approximation is not an approximation, but actually an equality. Of course, don’t be fooled into thinking that this means functions equal their quadratic approximations in general! The change of coordinates \(\Phi\) can be quite impossible to write down explicitly, so there is a lot of complexity buried in it. But the picture of the local behavior at a non-degenerate critical point is laid bare by this theorem. For degenerate critical points however, there is still a great deal of active research!

    3.

    The usual ‘Second Derivative Test’ offered up in a multivariable calculus textbook will look very confusing compared to our treatment. Firstly, the only functions considered will be of two variables \(f(x,y)\text{.}\) Then there will be a definition of the discriminant \(D\) of \(f\) at a critical point \(p\)
    \begin{equation*} D = \partial_x^2 f \partial_y^2 f - (\partial_x \partial_y f )^2. \end{equation*}
    Finally, the test will read:
    Theorem 1.5.13. Second Derivative Test.
    Suppose \(f(x,y)\) is defined in a neighborhood of \(p\) with continuous second partial derivatives. If \(p\) is a critical point and \(D \ne 0\) at \(p\) then
    1. \(D \gt 0\) and \(\partial^2_x f (p) \gt 0\) if and only if \(f\) achieves a local minimum at \(p\text{,}\)
    2. \(D \gt 0\) and \(\partial^2_x f (p) \lt 0\) if and only if \(f\) achieves a local maximum at \(p\text{,}\)
    3. if \(D \lt 0\) then \(f\) has a saddle point at \(p\text{.}\)
    Give an explanation of why this test is identical to Corollary 1.5.10 for the case of \(m = 2\text{.}\)
