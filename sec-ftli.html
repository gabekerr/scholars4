    Section 5.3 Stokesâ€™ Theorem on curves - The Fundamental Theorem of Line Integrals

    Our first application of Stokesâ€™ Theorem is for vector line integrals. First, we define a special type of vector field.

    Definition 5.3.1.

    A vector field \(\mb{F}\) is called conservative if there exists a function \(V\) on the space, called a potential for \(\mb{F}\text{,}\) for which
    \begin{equation*} \mb{F} = \nabla V. \end{equation*}
    The question of whether a vector field is conservative or not may seem difficult to answer, but there is a basic test that can be made which, in some cases, is enough.

    Proposition 5.3.2. Cross-partials test.

    If \(\mb{F} = F_1 \partial_1 + \cdots + F_n \partial_n\) is a conservative vector field on an open subset \(U\) of Euclidean space \(\mathbb{R}^n\) then
    \begin{equation*} \frac{\partial F_i}{\partial x_j} = \frac{\partial F_j}{\partial x_i} \end{equation*}
    for all \(1 \leq i, j \leq n\text{.}\)

    Proof.

    Exercise.
    The converse of this proposition is not always true, as we will see in a moment. However, if we add some conditions on the domain \(U\text{,}\) then the converse does hold.

    Example 5.3.3. Conservative vector field I.

    A vector field like
    \begin{equation*} \mb{F} = \left[ \begin{matrix} e^x \amp \cos y \end{matrix} \right] \end{equation*}
    whose coordinate functions are functions only of the given coordinate is always conservative. This is because we may integrate each function individually and add them to produce a potential
    \begin{equation*} V (x,y) = e^x + \sin y. \end{equation*}

    Example 5.3.4. Conservative vector field II.

    On the other hand, it may not be immediately clear that the vector field
    \begin{equation*} \mb{F} = \left[ \begin{matrix} 2xy - 1 \amp x^2 \end{matrix} \right] \end{equation*}
    is conservative. Here, we check to see that \(\partial_y (2x y - 1) = \partial_x (x^2)\text{.}\) This suggests that there may be a potential (whereas if the equality did not hold, we would know for certain that the field was not conservative). Integrating the first field with respect to \(x\) works straightaway and produces
    \begin{equation*} V (x,y) = x^2 y - x \end{equation*}
    as a potential.
    To prove the main theorem, we require a bit of vocabulary.

    Definition 5.3.5.

    A differential \(k\)-form \(\eta\) is called closed if \(\tder{}{\eta} = 0\) and exact if there is a \((k - 1)\)-form \(\xi\) for which \(\tder{}{\xi} = \eta\text{.}\)
    The definition of gradient is \(\nabla f = (\tder{}{f})^\vee\text{.}\) So another way of saying that a vector field is conservative is to say that it is the dual of an exact \(1\)-form. Let us now state the main theorem.

    Theorem 5.3.6. Fundamental Theorem of Line Integrals.

    Suppose \(\mathcal{C}\) is a piecewise linear curve in \(\mathbb{R}^n\) oriented from \(\mb{a}\) to \(\mb{b}\) and \(\mb{F}\) is a conservative vector field defined on a neighborhood of \(\mathcal{C}\) with potential \(V\text{.}\) Then
    \begin{equation*} \int_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}} = V (\mb{b}) - V (\mb{a} ). \end{equation*}

    Proof.

    Now, we recall that \(\nabla V = (\tder{}{V})^\vee\) and \(\tder{}{V}\) is an exact form. Also, dualizing twice does nothing, and so using the definition of vector line integral in DefinitionÂ 4.3.1 along with Stokesâ€™ TheoremÂ 5.2.7 we obtain
    \begin{align*} \int_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}} \amp = \int_{\mathcal{C}} \mb{F}^\vee, \\ \amp = \int_{\mathcal{C}} ( \nabla V)^\vee, \\ \amp = \int_{\mathcal{C}} ( (\tder{}{V})^\vee)^\vee, \\ \amp = \int_{\mathcal{C}} \tder{}{V}, \\ \amp = \int_{\partial \mathcal{C}} V, \\ \amp = \int_{\{(\mb{b}, +)\} \cup \{(\mb{a}, -)\}} V, \\ \amp = V (\mb{b} ) - V( \mb{a} ). \end{align*}
    An immediate corollary occurs here.

    Corollary 5.3.7.

    If \(\mb{F}\) is a conservative vector field and \(\mathcal{C}\) is a closed oriented curve then
    \begin{equation*} \oint_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}} = 0. \end{equation*}
    This follows from CorollaryÂ 5.2.9 and a similar argument as in the proof. It also gives us a very interesting counter-example to the converse of the cross-partials test.

    Example 5.3.8. A non-conservative vector field.

    Consider the vector field
    \begin{equation*} \mb{F} = -\frac{y}{x^2 + y^2} \partial_x + \frac{x}{x^2 + y^2} \partial_y \end{equation*}
    which is defined and infinitely differentiable in \(\mathbb{R}^2\) away from the origin. We can check that
    \begin{equation*} \frac{\partial}{\partial y} \left( -\frac{y}{x^2 + y^2} \right) = \frac{y^2 - x^2 }{(x^2 + y^2)^2} = \frac{\partial}{\partial x} \left( \frac{x}{x^2 + y^2} \right). \end{equation*}
    Thus \(\mb{F}\) satisfies the cross-partials test. On the other hand, If \(\mathcal{C}\) is the (closed) unit circle, oriented counterclockwise, then we computed in ExampleÂ 4.3.7 that
    \begin{equation*} \oint_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}} = 2 \pi. \end{equation*}
    Thus \(\mb{F}\) cannot be conservative even though it satisfies the cross-partials test!
    The last example should cause a bit of concern, because often we hope for an easy test to determine whether a given field has a potential. However, there is more to the story here.

    Proposition 5.3.9. Path Independence.

    Let \(\mb{F}\) be a vector field defined on an open set \(U\) of \(\mathbb{R}^n\text{.}\) Then \(\mb{F}\) is conservative if and only if the vector line integral
    \begin{equation*} \int_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}} \end{equation*}
    on a smooth connected oriented curve in \(U\) depends only on its initial and terminal points.

    Proof.

    If one assumes \(\mb{F}\) is conservative, then the fundamental theorem of line integrals immediately shows it is path independent. On the other hand, if it is path independent, for each connected component \(W\) of \(U\text{,}\) we choose a point \(\mb{a} \in W\) and define the function
    \begin{equation} V (\mb{x} ) = \int_{P (\mb{x} )} \mb{F} \cdot \tder{}{\mb{r}}\tag{5.3.1} \end{equation}
    where \(P ( \mb{x})\) is any path from \(\mb{a}\) to \(\mb{x}\text{.}\) Showing \(V\) is indeed a potential for \(\mb{F}\) is left as an exercise.
    What is meant here is that if \(\tilde{\mathcal{C}}\) is any other connected curve in \(U\) with the same initial and terminal points, then
    \begin{equation*} \int_{\tilde{\mathcal{C}}} \mb{F} \cdot \tder{}{\mb{r}} = \int_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}}. \end{equation*}
    This property of \(\mb{F}\) is called path independence.
    The last proposition should be thought of as a way to characterize conservative vector fields. It certainly is not something that can be verified through trial and error because you would need to calculate a vector line integral for every path between points (of which there are infinitely many). However, we can use this result to a great purpose, but we need a definition first. We call a closed curve \(\mathcal{C}\) a simple closed curve if there is a 1-1 continuous map from the circle to \(\mathcal{C}\text{.}\)

    Definition 5.3.10.

    A domain \(E\) in \(\mathbb{R}^n\) is called simply connected if, given any piecewise smooth simple closed curve \(\mathcal{C}\) in \(E\text{,}\) there is a smooth map from the unit disc \(D\) to \(E\) whose boundary is \(\mathcal{C}\text{.}\)
    One should think of lassos or rubber bands when considering whether a region is simply connected. Imagine the domain \(E\) has a lasso thrown in it and the person wants to tighten the lasso to a point. Can they do it while keeping it in \(E\text{?}\) If the answer is always yes, then \(E\) is simply connected, otherwise, it is not. The upshot of this definition is that many domains are in fact simply connected, and when they are, we have the following very helpful result!
    Before proving the next proposition, we make a note that the cross-partials condition on \(\mb{F}\) is precisely the condition that
    \begin{equation*} \tder{}{\mb{F}^\vee} = \sum_{1 \leq i \lt j \leq n} \left( \frac{\partial F_j}{\partial x_i} - \frac{\partial F_i}{\partial x_j} \right) \, \tder{}{x_i} \wedge \tder{}{x_j} = 0. \end{equation*}
    Or, using our new vocabulary, that the dual \(\mb{F}^\vee\) is a closed \(2\)-form.

    Proposition 5.3.11.

    Suppose \(\mb{F}\) is a vector field defined on a simply connected open set \(U\) of \(\mathbb{R}^n\text{.}\) Then \(\mb{F}\) is conservative if and only if it satisfies the cross-partials test.

    Proof.

    By PropositionÂ 5.3.2 and PropositionÂ 5.3.9 we need only show that if \(\mb{F}\) satisfies the cross-partials test then it is path independent. Suppose \(C_1\) and \(C_2\) are two smooth paths from \(\mb{a}\) to \(\mb{b}\) in \(U\text{.}\) If they intersect, we may break them up into shorter paths where they do not (or replace one of them over small intervals by a new set of paths). So let us assume they only intersect at the endpoints. Then, reversing the orientation of \(C_2\) we may glue these two paths together to obtain a simple oriented closed curve \(\mathcal{C} = C_1^+ \cup C_2^{-}\) (here the sign in the exponent indicates a change in orientation). Since \(U\) is simply connected, there is a smooth map \(\mb{G} : D \to U\) which sends the boundary \(S^1\) of \(D\) to \(\mathcal{C}\text{.}\) But then, using the fact that \(\mb{F}^\vee\) is closed since it passes the cross-partials test and Stokesâ€™ TheoremÂ 5.2.7 we obtain
    \begin{align*} \int_{C_1} \mb{F} \cdot \tder{}{\mb{r}} - \int_{C_2} \mb{F} \cdot \tder{}{\mb{r}} \amp = \int_{\mathcal{C}} \mb{F} \cdot \tder{}{\mb{r}}, \\ \amp = \int_{\mathcal{C}} \mb{F}^\vee , \\ \amp = \int_{S^1} \mb{G}^* \mb{F}^\vee ,\\ \amp = \int_{\partial D} \mb{G}^* \mb{F}^\vee , \\ \amp = \int_{D} \tder{}{\left(\mb{G}^* \mb{F}^\vee \right)} , \\ \amp = \int_{D} \mb{G}^*\left( \tder{}{ \mb{F}^\vee} \right), \\ \amp = 0. \end{align*}
    Thus
    \begin{equation*} \int_{C_1} \mb{F} \cdot \tder{}{\mb{r}} = \int_{C_2} \mb{F} \cdot \tder{}{\mb{r}} \end{equation*}
    and \(\mb{F}\) is path independent.
    Taking another look at ExampleÂ 5.3.8 we can identify the culprit behind why \(\mb{F}\) is not conservative. Namely, \(\mb{F}\) is defined on the punctured plane, which is not simply connected. However, changing the domain of \(\mb{F}\) so that it is all points in the plane except the positive \(x\)-axis would in fact make \(\mb{F}\) conservative!
