    Section 5.1 Derivatives

    Having discussed all of the different types of integrals of interest to us, we return for a moment to derivatives.

    Subsection 5.1.1 Exterior derivative

    The exterior derivative is not only a derivative of functions, but of forms themselves. For the remainder of the course, we will assume all forms have coefficient functions that have continuous second partial derivatives.

    Definition 5.1.1.

    Let
    \begin{equation*} \eta = \sum_{1 \leq i_1 \lt \cdots \lt i_m \leq n} f_{i_1, \ldots ,i_m } \, \tder{}{x_{i_1}} \wedge \cdots \wedge \tder{}{x_{i_m}} \end{equation*}
    be a differential \(m\)-form in \(\mathbb{R}^n\text{.}\) The exterior derivative of \(\eta\) is the \((m + 1)\)-form
    \begin{equation*} \tder{}{\eta} = \sum_{1 \leq i_1 \lt \cdots \lt i_m \leq n} \tder{}{f_{i_1, \ldots ,i_m }} \wedge \tder{}{x_{i_1}} \wedge \cdots \wedge \tder{}{x_{i_m}} \end{equation*}
    Let us compute a few examples.

    Example 5.1.2. Exterior derivative of a scalar function.

    For a differentiable function \(f\) on an open set \(U\text{,}\) which is a \(0\)-form by definition, of \(\mathbb{R}^n\text{,}\) we recall
    \begin{equation*} \tder{}{f} = \frac{\partial f}{\partial x_1} \tder{}{x_1} + \cdots + \frac{\partial f{\partial x_n} \tder{}{x_n} . \end{equation*}
    So, for example if \(f(x,y) = x \cos (xy)\) then
    \begin{equation*} \tder{}{f} = \left( \cos (xy) - xy \sin (xy) \right) \tder{}{x} - x^2 \sin (xy) \, \tder{}{y}. \end{equation*}
    This is just one way of expressing the derivative of a scalar function.

    Example 5.1.3. Exterior derivative of a \(1\)-form.

    Now suppose we have a one form in \(\mathbb{R}^3\) like
    \begin{equation*} \eta = x \tder{}{x} + y \tder{}{y} + z \tder{}{z}. \end{equation*}
    Then
    \begin{align*} \tder{}{\eta} \amp = \tder{}{} \left( x \tder{}{x} - z \tder{}{y} + y \tder{}{z} \right), \\ \amp = \tder{}{} \left( x \tder{}{x} \right) - \tder{}{} \left(z \tder{}{y} \right) + \tder{}{} \left( y \tder{}{z} \right), \\ \amp = \tder{}{x} \wedge \tder{}{x} - \tder{}{z} \wedge \tder{}{y} + \tder{}{y} \wedge \tder{}{z}, \\ \amp = 2 \, \tder{}{y} \wedge \tder{}{z}. \end{align*}

    Example 5.1.4. Exterior derivative of a \(2\)-form.

    For the \(2\)-form
    \begin{equation*} \xi \amp = xz \, \tder{}{x} \wedge \tder{}{y} - xy \, \tder{}{y} \wedge \tder{}{z} \end{equation*}
    one calculates
    \begin{align*} \tder{}{\xi} \amp = \tder{}{} \left( xz \, \tder{}{x} \wedge \tder{}{y} - xy \, \tder{}{y} \wedge \tder{}{z} \right), \\ \amp = \tder{}{(xz)} \wedge \tder{}{x} \wedge \tder{}{y} - \tder{}{xy} \wedge \tder{}{y} \wedge \tder{}{z}, \\ \amp = \left( x \tder{}{z} + z \tder{}{x} \right) \wedge \tder{}{x} \wedge \tder{}{y} - \left( x \tder{}{y} + y \tder{}{x} \right) \wedge \tder{}{y} \wedge \tder{}{z}, \\ \amp = x \, \tder{}{z} \wedge \tder{}{x} \wedge \tder{}{y} - y\, \tder{}{x} \wedge \tder{}{y} \wedge \tder{}{z}, \\ \amp = (x - y)\, \tder{}{x} \wedge \tder{}{y} \wedge \tder{}{z}. \end{align*}

    Example 5.1.5. Exterior derivative of a top form.

    Note that for a top form, we always have that its exterior derivative is zero. For example, if \(\eta = xy \tder{}{x} \wedge \tder{}{y}\) in \(\mathbb{R}^2\text{,}\) then
    \begin{equation*} \tder{}{\eta} = \left( x \tder{}{y} + y \tder{}{x} \right) \wedge \tder{}{x} \wedge \tder{}{y} = 0. \end{equation*}
    There are a few important properties about the exterior derivative that make it easier to work with.

    Proposition 5.1.6.

    The exterior derivative satisfies the following properties.
    Naturality If \(\mb{G} : U \to V\) is a differentiable map from an open set \(U \subset \mathbb{R}^m\) to \(V \subset \mathbb{R}^n\) and \(\eta\) is a differential \(k\)-form on \(U\) then
    \begin{equation*} \mb{G}^* \tder{}{\eta} = \tder{}{\mb{G}^* \eta}. \end{equation*}
    Linearity If \(a\) and \(b\) are real numbers, \(\eta_1\) and \(\eta_2\) are \(m\)-forms then
    \begin{equation*} \tder{}{(a \eta_1 + b \eta_2)} = a \tder{}{\eta_1} + b \tder{}{\eta_2}, \end{equation*}
    Leibniz Rule For an \(m\)-form \(\eta\) and \(n\)-form \(\xi\)
    \begin{equation*} \tder{}{\left( \eta \wedge \xi \right)} = \tder{}{\eta} \wedge \xi + (-1)^m \eta \wedge \tder{}{\xi}, \end{equation*}
    Square Zero For any \(m\)-form \(\eta\text{,}\) \(\tder{}{} \left( \tder{}{\eta} \right) = 0\text{.}\)

    Proof.

    We leave the first two properties as exercises. For the last property, we first check this for a scalar function \(f\) (i.e. a \(0\)-form).
    \begin{align*} \tder{}{} \left( \tder{}{f} \right) \amp = \tder{}{} \left( \frac{\partial f}{\partial x_1} \tder{}{x_1} + \cdots + \frac{\partial f}{\partial x_n} \tder{}{x_n} \right), \\ \amp = \tder{}{} \left( \frac{\partial f}{\partial x_1} \tder{}{x_1} + \cdots + \frac{\partial f}{\partial x_n} \tder{}{x_n} \right), \\ \amp = \sum_{j = 1}^n \tder{}{} \left( \frac{\partial f}{\partial x_j} \tder{}{x_j} \right), \\ \amp = \sum_{i = 1}^n \sum_{j = 1}^n \frac{\partial^2 f}{\partial x_i \partial x_j} \, \tder{}{x_i} \wedge \tder{}{x_j}. \end{align*}
    Now, for all summands in the last expression with \(i = j\text{,}\) we have \(\tder{}{x_i} \wedge \tder{}{x_j} = 0\text{,}\) so these vanish. The remaining summands come in pairs and we obtain
    \begin{align*} \tder{}{} \left( \tder{}{f} \right) \amp = \sum_{1 \leq i \lt j \leq n} \left( \frac{\partial^2 f}{\partial x_i \partial x_j} \, \tder{}{x_i} \wedge \tder{}{x_j} + \frac{\partial^2 f}{\partial x_j \partial x_i} \, \tder{}{x_j} \wedge \tder{}{x_i} \right), \\ \amp = 0 . \end{align*}
    Here we used our blanket assumption that \(f\) has continuous second partial derivatives and then applied Clairaut’s Theorem. For \(m\)-forms with \(m \gt 0\text{,}\) we may use linearity and relabel the indices of the variables (if needed) and verify the property for
    \begin{equation*} \eta = f \, \tder{}{x_1} \wedge \cdots \tder{}{x_m}. \end{equation*}
    Then, using the Leibniz rule for forms, we obtain
    \begin{align*} \tder{}{} \left( \tder{}{\eta} \right) \amp = \tder{}{} \left( \tder{}{f} \wedge \tder{}{x_1} \wedge \cdots \tder{}{x_m}\right) , \\ \amp = \left( \tder{}{} \tder{}{f} \right) \wedge \tder{}{x_1} \wedge \cdots \tder{}{x_m} - \tder{}{f} \wedge \tder{}{} \left( \tder{}{x_1} \wedge \cdots \tder{}{x_m} \right), \\ \amp = 0. \end{align*}
    These properties have led mathematicians and physicists to consider the vector space of all differential forms on some \(m\)-dimensional open subset \(U\) of \(\mathbb{R}^m\) (and more generally, a smooth \(m\)-dimensional subspace \(X\)). They write \(\Omega^k (U)\) for the vector space of differential \(k\)-forms and
    \begin{equation*} \Omega^* (U) = \bigoplus_{k = 0}^m \Omega^k (U) \end{equation*}
    for the vector space of all differential forms on \(U\text{.}\) Then the exterior derivative leads to a sequence of linear maps
    \begin{equation*} \Omega^0 (U) \stackrel{\tder{}{}}{\longrightarrow} \Omega^1 (U) \stackrel{\tder{}{}}{\longrightarrow} \cdots \stackrel{\tder{}{}}{\longrightarrow} \Omega^{m - 1} (U) \stackrel{\tder{}{}}{\longrightarrow} \Omega^{m } (U) \end{equation*}
    where the composition of any two of these is identically zero. Note that \(\Omega^* (U)\) also has a product given by wedging forms together. Taken altogether, \((\Omega^* (U), \tder{}{}, \wedge )\) is called the de Rham complex and has occupied a great deal of attention for the past \(90\) years. While a difficult object to understand at first, it unifies many previously independent theorems into one cohesive framework and thereby amplifies potential applications.

    Subsection 5.1.2 Gradient

    We have already seen the definition that the gradient of a function \(f\) on an open set \(U\) in Euclidean space \(\mathbb{R}^n\) is the vector field
    \begin{equation*} \nabla f = \frac{\partial f}{\partial x_1} \partial_{x_1} + \cdots + \frac{\partial f}{\partial x_n} \partial_{x_n}. \end{equation*}
    While it is of no great surprise, we must record the following simple proposition.

    Proposition 5.1.7.

    For a scalar function \(f\) on an open subspace of Euclidean space,
    \begin{equation} \left( \nabla f \right)^\vee = \tder{}{f} .\tag{5.1.1} \end{equation}
    In words, the dual of the gradient is the exterior derivative of \(f\text{.}\)
    In fact, in more advanced settings where one must use a varying inner product because the space being dealt with is not flat, equation (5.1.1) gives an alternative definition of the gradient.
    As we have already studied the main properties of the gradient, we leave this as a reminder that such a derivative exists which takes a scalar field to a vector field so can be considered as a map
    \begin{equation*} \text{Scalars} (U) \stackrel{\nabla}{\longrightarrow} \text{Vect} (U) \end{equation*}
    where \(\text{Scalars} (U)\) is the vector space of scalar functions on \(U\) and \(\text{Vect} (U)\) is the vector space of vector fields.

    Subsection 5.1.3 Curl

    The next derivative we consider is rather unique and only occurs in dimension \(3\) (although there is something called \(2\)-dimensional curl which we will discuss later in this section).

    Definition 5.1.8.

    For a differentiable vector field
    \begin{equation*} \mb{F} = \left[ \begin{matrix} P \amp Q \amp R \end{matrix} \right] \end{equation*}
    defined on an open subset \(U\) of \(\mathbb{R}^3\text{,}\) the curl of \(\mb{F}\) is the vector field
    \begin{equation*} \text{curl} ( \mb{F} ) = \left[ \begin{matrix} \left( \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} \right) \amp \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} \right) \amp \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) \end{matrix} \right]. \end{equation*}
    This rather cryptic definition has an alternative notation that is more suggestive of the formula
    \begin{equation*} \text{curl} (\mb{F} ) = \nabla \times \mb{F}. \end{equation*}
    Let us compute an example to see how this helps.

    Example 5.1.9. Curl computation.

    Let
    \begin{equation*} \mb{F} = \left[ \begin{matrix} yz - z \amp xz \amp xy + y \end{matrix} \right]. \end{equation*}
    We use the suggestive formula to compute its curl
    \begin{align*} \text{curl} (\mb{F} ) \amp = \nabla \times \mb{F}, \\ \amp = \det \left[ \begin{matrix} \mb{e}_1 \amp \mb{e}_2 \amp \mb{e}_3 \\ \partial_x \amp \partial_y \amp \partial_z \\ yz -z \amp xz \amp xy + y \end{matrix} \right], \\ \amp = \left[ \begin{matrix} ((x + 1) - x ) \amp ((y - 1) - y) \amp (z - z) \end{matrix} \right], \\ \amp = \left[ \begin{matrix} 1 \amp -1 \amp 0 \end{matrix} \right]. \end{align*}
    The geometric interpretation of curl is that it measures the infinitesimal rotation of the vector field \(\mb{F}\text{.}\) In particular, the vector \(\text{curl} (\mb{F} )_p\) at \(p \in U\) gives the axis of rotation that the field is rotating about and its magnitude expresses the strength of the rotation.
    Note that curl takes vector fields to vector fields, so we can think of it as a linear map
    \begin{equation*} \text{Vect} (U) \stackrel{\text{curl}}{\longrightarrow} \text{Vect} (U) \end{equation*}
    In fact, we can connect this new derivative with the exterior derivative as follows.

    Proposition 5.1.10.

    For a differentiable vector field \(\mb{F}\) defined on an open subset \(U\) of \(\mathbb{R}^3\) endowed with the dot product and standard orientation,
    \begin{equation*} \text{curl} (\mb{F} ) = \left[ \star \left( \tder{}{\mb{F}^\vee} \right) \right]^\vee. \end{equation*}
    In other words, the curl of \(\mb{F}\) is the dual vector field of the Hodge star of the exterior derivative of the dual \(1\)-form of \(\mb{F}\text{!}\)

    Subsection 5.1.4 Two dimensional curl

    There is a version of curl for \(2\) dimensions.

    Definition 5.1.11.

    Let \(\mb{F} = P \partial_x + Q \partial_y\) be a differentiable vector field on a region in \(\mathbb{R}^2\text{.}\) The two dimensional curl of \(\mb{F}\) is the scalar function
    \begin{equation*} \text{curl}^{2d} (\mb{F} ) = \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} . \end{equation*}
    Note that two dimensional curl takes vector fields to scalar functions, so we can think of it as a linear map
    \begin{equation*} \text{Vect} (U) \stackrel{\text{curl}^{2d}}{\longrightarrow} \text{Scalars} (U) \end{equation*}
    The two dimensional curl is a version of the curl in the sense that if we think of \(\mb{F}\) as a vector field \(\tilde{\mb{F}}\) on \(U \times \mathbb{R} \subset \mathbb{R}^3\) then
    \begin{equation*} \text{curl} (\tilde{\mb{F}}) = \left[ \begin{matrix} 0 \amp 0 \amp \text{curl}^{2d} (\mb{F} ) \end{matrix} \right]. \end{equation*}
    Again, this derivative can be related to the exterior derivative.

    Proposition 5.1.12.

    Let \(\mb{F}\) be a differentiable vector field defined on an open set \(U\) in the Euclidean plane \(\mathbb{R}^2\) equipped with the dot product and the standard orientation. Then
    \begin{equation*} \text{curl}^{2d} (\mb{F} ) = \star \tder{}{} \left( \mb{F}^\vee \right) \end{equation*}
    The interpretation of two dimensional curl is that it gives a scalar measure of the infinitesimal counter-clockwise rotation of the field at a point.

    Subsection 5.1.5 Divergence

    The last derivative we consider takes a vector field and produces a scalar function on a domain \(U\) in \(\mathbb{R}^n\text{.}\)

    Definition 5.1.13.

    Let
    \begin{equation*} \mb{F} = f_1 \partial_1 + \cdots + f_n \partial_n \end{equation*}
    be a differentiable vector field defined on an open set \(U\) in Euclidean space \(\mathbb{R}^n\text{.}\) The divergence of \(\mb{F}\) is the function
    \begin{equation*} \text{div} (\mb{F} ) = \frac{\partial f_1}{\partial x_1} + \cdots + \frac{\partial f_n}{\partial x_n}. \end{equation*}
    While the formula for the divergence is much easier to remember, there is still an alternative suggestive notation
    \begin{equation*} \text{div} (\mb{F} ) = \nabla \cdot \mb{F}. \end{equation*}
    Let’s compute an example.

    Example 5.1.14. Divergence computation.

    Suppose \(\mb{F} = x \partial_x + yx \partial_y\) in \(\mathbb{R}^2\text{.}\) Then
    \begin{equation*} \text{div} (\mb{F} ) = \frac{\partial}{\partial x} ( x) + \frac{\partial}{\partial y} (yx) = 1 + x. \end{equation*}
    The divergence of a vector field is a measure of how much the vector field spreads or collapses at a given point. This viewpoint will become especially clear after the Divergence Theorem.
    Note that divergence takes vector fields to scalar functions, so we can think of it as a linear map
    \begin{equation*} \text{Vect} (U) \stackrel{\text{div}}{\longrightarrow} \text{Scalars} (U) \end{equation*}
    Relating this derivative to forms can be done as follows.

    Proposition 5.1.15.

    Let \(\mb{F}\) be a differentiable vector field defined on an open set \(U\) in Euclidean space \(\mathbb{R}^n\) equipped with the dot product and the standard orientation. Then
    \begin{equation*} \text{div} (\mb{F} ) = \star \tder{}{} \left(\star \mb{F}^\vee \right) \end{equation*}
    Again we see that our ‘new’ derivative is but a shadow of the exterior derivative (along with the linear algebra of dualizing and Hodge stars).
