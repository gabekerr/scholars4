    Section 1.1 The limit

    In this section, we will review limits of vector valued functions of many variables
    \begin{equation*} \mb{F} : \mathbb{R}^m \to \mathbb{R}^n. \end{equation*}
    Of course, we can and should be more general than this. We should consider a vector \(\mb{c}\) of an inner product space \(V\) in an open set \(U\) and really work there. The reason for all of this additional notation is that sometimes our function \(\mb{F}\) may only be defined near a given point. Indeed, the limit and derivative are ‘local’ features of a function meaning they do not care about what is going on far away from the point of focus.
    To understand the definition of the limit, one must work through definitions and results in a basic analysis course. As this is not in the main body of the notes, we will review a few of the important features and discuss examples in this section.

    Definition 1.1.1.

    Let \(V_1\) and \(V_2\) be two inner product spaces and \(U \subseteq V_1\) a neighborhood of \(\mb{c}\text{.}\) Suppose
    \begin{equation*} f : U \to V_2 \end{equation*}
    is any function. We say that the limit of \(f(\mb{x})\) as \(\mb{x}\) approaches \(\mb{c}\) is \(\mb{L}\) if for every \(\varepsilon \gt 0\) there is a \(\delta \gt 0\) so that
    \begin{equation*} 0 \lt \|\mb{x} - \mb{c} \| \lt \delta \end{equation*}
    implies
    \begin{equation*} \| f(\mb{x}) - \mb{L} \| \lt \varepsilon . \end{equation*}
    In this case, we write
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} f(\mb{x} ) = \mb{L} . \end{equation*}
    An alternative way of stating the implication is that every point in the punctured ball of radius \(\delta\) around \(\mb{c}\) is sent to the ball of radius \(\varepsilon\) around \(\mb{L}\) or, with notation,
    \begin{equation*} \mb{x} \in \pball{\delta}{\mb{c}} \text{ implies } f(\mb{x} ) \in \ball{\varepsilon}{\mb{L}}. \end{equation*}
    Invariably, with the notion of limit comes its cousin.

    Definition 1.1.2.

    Let \(V_1\) and \(V_2\) be two inner product spaces and \(U \subseteq V_1\) a neighborhood of \(\mb{c}\text{.}\) Suppose
    \begin{equation*} f : U \to V_2 \end{equation*}
    is any function. We say that \(f\) is continuous at \(\mb{c}\) if
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} f(\mb{x} ) = f ( \mb{c} ) . \end{equation*}
    The definition given here for limits allows us to talk about limits for non-linear functions between inner product spaces. It is interesting to consider what exactly has become new or challenging compared to our one-dimensional understanding. In fact, there are several results that have much the same look in higher dimensions. The case where this is most obvious is scalar functions. Recall these are functions defined on an open subset \(U\) of an inner product space \(V\) to \(K\) (where \(K\) is \(\mathbb{R}\) or \(\mathbb{C}\)). In notation we would write \(f : U \to K\text{.}\) The reason that scalar functions are such important creatures is that we can add, multiply and divide them as well as compose them with some familiar functions from \(K\) to \(K\text{.}\)

    Proposition 1.1.3.

    Suppose \(U\) is an open set in \(V\text{,}\) \(\lambda\) a scalar in \(K\text{,}\) \(h: K \to K\) is a continuous function at \(L\text{,}\) and \(f, g : U \to K\) are scalar functions such that
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} f(\mb{x} ) = L, \hspace{.1in} \lim_{\mb{x} \to \mb{c}} g(\mb{x} ) = M . \end{equation*}
    Then
    1. \begin{equation*} \lim_{\mb{x} \to \mb{c}} (\lambda f + g) (\mb{x} ) = \lambda L + M \text{,} \end{equation*}
    2. \begin{equation*} \lim_{\mb{x} \to \mb{c}} f (\mb{x} ) \, g(\mb{x}) = L M , \end{equation*}
    3. if \(M \ne 0\text{,}\)
      \begin{equation*} \lim_{\mb{x} \to \mb{c}} \frac{f (\mb{x} )}{g (\mb{x} )} = \frac{L}{M}. \end{equation*}
    4. \begin{equation*} \lim_{\mb{x} \to \mb{c}} (h \circ f)(\mb{x} ) = h (L). \end{equation*}
    We also need the following ‘base case’ result.

    Proposition 1.1.4.

    A linear scalar function on a finite dimensional inner product space is continuous.

    Proof.

    Let \(V\) be an inner product space over \(K\) and \(\alpha : V \to K\) a linear function. Let \(W = \ker ( \alpha )\) and \(W^\perp\) its orthogonal complement. There are two cases for \(\alpha\text{.}\) First, \(\alpha\) could be the zero function \(\alpha (\mb{v} ) = 0\) in which case it is easy to see that it is continuous (why?). Otherwise, restricting \(\alpha\) to \(W^\perp\) is a non-zero linear function on a one-dimensional space. Thus, choosing a unit vector \(\mb{w}_1\) in \(W^\perp\) (and therefore a basis), we have
    \begin{equation*} \alpha ( \mb{w}_1 ) = a \ne 0 . \end{equation*}
    Now, given any \(\varepsilon \gt 0\text{,}\) let \(\delta = \frac{\varepsilon}{|a|}\) and suppose
    \begin{equation*} \| \mb{x} - \mb{c} \| \lt \delta = \frac{\varepsilon}{|a|} \end{equation*}
    Then we can use projections, the unique decomposition of vectors in \(V = W \oplus W^\perp\) and the Cauchy-Schwarz Inequality to see
    \begin{align*} | \alpha ( \mb{x} ) - \alpha ( \mb{c} ) | \amp = | \alpha ( \mb{x} - \mb{c} ) |, \\ \amp = \left| \alpha \left( \text{proj}_W ( \mb{x} - \mb{c} ) + \text{proj}_{W^\perp} ( \mb{x} - \mb{c} ) \right) \right|, \\ \amp = \left| \alpha \left( \text{proj}_W ( \mb{x} - \mb{c} ) \right) + \alpha \left( \text{proj}_{W^\perp} ( \mb{x} - \mb{c} ) \right) \right|, \\ \amp = \left| \alpha \left( \langle \mb{x} - \mb{c} , \mb{w}_1 \rangle \mb{w}_1 \right) \right|, \\ \amp = \left| \langle \mb{x} - \mb{c} , \mb{w}_1 \rangle \alpha (\mb{w}_1 ) \right|, \\ \amp = \left| \langle \mb{x} - \mb{c} , \mb{w}_1 \rangle\right| \, \left|\alpha (\mb{w}_1 ) \right| , \\ \amp \leq \| \mb{x} - \mb{c} \| \, \| \mb{w}_1 \| |a|, \\ \amp \lt \frac{\varepsilon}{|a|} | a| = \varepsilon. \end{align*}
    The point of the last proposition is that we can use it, along with Proposition 1.1.3 to compute many continuous scalar functions.

    Example 1.1.5. Building continuous function I.

    If \(Q\) is a quadratic form on \(\mathbb{R}^n\text{,}\) then we know it can be written as a sum
    \begin{equation*} Q (x_1, x_2, \ldots, x_n) = \sum_{i = 1}^n \sum_{j = 1}^n a_{ij} x_i x_j . \end{equation*}
    By Proposition 1.1.4, each of the coordinate functions \(x_i\) are continuous. But then so is \(Q\) as it is a linear combination of products of such functions.

    Example 1.1.6. Building continuous function II.

    Proposition 1.1.4 tells us that the functions \(x, y\) and \(z\) are continuous functions from \(\mathbb{R}^3\) to \(\mathbb{R}\text{.}\) This may seem trivial, but using our arithmetic theorems and knowledge of continuous functions on \(\mathbb{R}\) like \(\sin, \ln\) and \(\exp\text{,}\) we can build many continuous functions. For example, we have that
    \begin{equation*} f(x,y,z) = \sin \left(z e^{x + y} \right) - \frac{\ln (|z| + 1)}{x^2 + y^2 + 1} \end{equation*}
    is a continuous function on all of \(\mathbb{R}^3\) as it is an arithmetic combination of well defined compositions of continuous functions.
    The limit of a vector valued function can be found by the limits of its scalar coordinate functions (if they exist), so we will restrict our attention to scalar functions
    \begin{equation*} f : \mathbb{R}^m \to \mathbb{R}. \end{equation*}
    As a large scale goal, we would like to know when
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} f(\mb{x} ) \end{equation*}
    exists and how to calculate it for a given point \(\mb{c}\text{.}\) Now, if \(f\) is continuous, then we can simply put in \(\mb{c}\) and move along. However, just as in one variable calculus, the most interesting examples are when \(f\) is undefined at \(\mb{c}\text{.}\) In these cases, we have the usual one variable issues of what can go wrong, but with the massively more complicated context of being in higher dimensions.

    Example 1.1.7. Issue 1 - Divergence.

    If we consider the function
    \begin{equation*} f(x) = \frac{1}{x^2} \end{equation*}
    and the limit \(\lim_{x \to 0} f(x)\text{,}\) we all know that this limit does not exist ... or we’ve been informed that the limit is \(\infty\text{.}\) Either is fine, what is important is that the phenomenon is clear : as \(x\) tends to \(0\text{,}\) the value of the function will pass any finite bound. Similar situations occur in functions of many variables as well. For example,
    \begin{equation*} f(x, y) = \frac{1}{x^2 + y^2} \end{equation*}
    also tends to infinity as \((x,y)\) approaches \((0,0)\text{.}\) In these situations, it will be acceptable to say that the limit does not exist (although it is fine if you wish to be nuanced and say it is \(\infty\)).

    Example 1.1.8. Issue 2 - Oscillation.

    Another strange thing that can happen with functions is that they jump around a good amount near the limiting point. For example,
    \begin{equation*} f(x) = \sin \left( \frac{1}{x} \right) \end{equation*}
    oscillates more and more as you approach \(0\text{.}\) Here, there is no getting around the limit not existing and we write DNE with great confidence. Of course, similar oscillation can occur in higher dimensions, simply throw your favorite function into the one above. For example,
    \begin{equation*} f(x,y,z) = \sin \left( \frac{1}{x^2 + y^2 + z^2} \right) \end{equation*}
    will lead to
    \begin{equation*} \text{ The limit } \lim_{(x,y,z) \to (0, 0, 0)} f(x,y,z) \text{ does not exist.} \end{equation*}

    Example 1.1.9. Issue 3 - Left limit not equal to right limit.

    Here, the universe breaks. In fact, this is where, if we think about it enough, we start to appreciate how easy life was in one variable calculus. Let’s recall that we can define functions like the Heavyside function
    \begin{equation} H (x) = \begin{cases} 0 \amp x \leq 0, \\ 1 \amp x \gt 0. \end{cases}\tag{1.1.1} \end{equation}
    Then our one sided limits are
    \begin{equation*} \lim_{x \to 0^-} H(x) = 0 \ne 1 = \lim_{x \to 0^+} H(x). \end{equation*}
    We conclude that the limit does not exist because it should indeed be the same value.
    In higher dimensions, how do we phrase this? Indeed, in one dimension for any given point there is a left and a right approach and that is all. However, in higher dimensions there are an infinite number of approaches... and even more than you think! Let’s look at an example to see this. Let
    \begin{equation*} f(x,y) = \frac{xy}{x^2 + y^2} . \end{equation*}
    Then in fact \(\lim_{(x,y) \to (0, 0)} f(x,y)\) does not exist. We could check this by approaching the origin in various directions. For example, we could approach along the \(x\)-axis \((t, 0)\) and see
    \begin{equation*} \lim_{t \to 0} f(t,0) = \frac{0}{t^2 + 0} = 0. \end{equation*}
    But then we could approach along the diagonal line \((t,t)\) and see
    \begin{equation*} \lim_{t \to 0} f(t, t) = \frac{t^2}{t^2 + t^2} = \frac{1}{2}. \end{equation*}
    Since these two limits are not equal, we can conclude that the limit does not exist. As a rule of thumb, if you consider a rational function
    \begin{equation*} f(x_1, \ldots, x_n) = \frac{P(x_1, \ldots, x_n)}{Q(x_1, \ldots, x_n)} \end{equation*}
    where the degree of the homogeneous numerator equals that of the homogeneous denominator, you should expect to find this behavior.
    The last example is one that should be considered with great care. Unlike in \(1\) dimension, it is not the case that one can assert that a limit does exist if all straight line limits exist and are equal.

    Example 1.1.10. Non-existent limit with well defined limits along straight lines I.

    Consider the function
    \begin{equation*} f(x,y) = \begin{cases} 0 \amp y \ne x^2 , \\ 1 \amp y = x^2. \end{cases} \end{equation*}
    One can easily check that \(\lim_{t \to 0} f(at, bt) = 0\) for any real numbers \(a\) and \(b\text{.}\) However, \(\lim_{(x,y) \to (0, 0)} f(x,y)\) does not exist. Indeed, this shows that straight line approaches are not enough, one must also consider parabolic approaches, and of course, an infinite set of other approaches!

    Example 1.1.11. Non-existent limit with well defined limits along straight lines II.

    The last example may make it seem that only very exotic looking functions experience this defect. To see how common such a scenario may be, consider the function
    \begin{equation*} f(x,y) = \frac{x^8}{(y - x^2)^2 + y^4} \end{equation*}
    which is continuous everywhere except at the origin where it is undefined. We again consider the limit of \(f(x,y)\) as one approaches \((0, 0)\text{.}\) One could try to examine the limit of \(f(x,y)\) along any line \((t, at)\) through the origin (the case of \((0,t)\) can be evaluated separately). Here we have
    \begin{equation*} f(t, at) = \frac{t^8}{(at - t^2)^2 + a^4t^4}= \frac{t^6}{(a - t)^2 + a^4 t^2} \end{equation*}
    which, if \(a \ne 0\) converges to \(0\text{.}\) The case of \(a = 0\) gives \(f(t, 0) = t^4\) which converges to \(0\) as well. It may than seem that the limit of \(f(x,y)\) as \((x,y) \to (0,0)\) is zero. After all, we have approached the origin in every linear direction and found the same limit of \(0\text{.}\) However, the limit in fact does not exist! Indeed, if we approach the origin along the parabola, \((t, t^2)\) we see
    \begin{equation*} f(t,t^2) = \frac{t^8}{(t^2 - t^2)^2 + (t^2)^4} = \frac{t^8}{t^8} = 1 \end{equation*}
    so that the limit is \(1\) on this approach. This phenomena shows that, while taking different linear approaches may show that a limit does not exist, it cannot show that a limit does exist.
    This last example may leave the student a bit concerned about how in the world they will prove that certain limits exist. Such a concern is indeed warranted as there are fewer tools in many variables (for example, L’Hopital’s rule does not exist in many variables). However, there is a higher dimensional Squeeze Theorem that comes in handy which is proven in the appendix.

    Theorem 1.1.12. Squeeze Theorem.

    Suppose \(U\) is an open neighborhood of \(\mb{c}\) in an inner product space \(V\) and \(g : U \to \mathbb{R} \) is a scalar valued function. If \(f : U \to \mathbb{R}\) and \(h: U\to \mathbb{R}\) are functions for which
    \begin{equation*} f(\mb{x}) \leq g(\mb{x} ) \leq h(\mb{x}), \end{equation*}
    and
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} f(\mb{x}) = L = \lim_{\mb{x} \to \mb{c}} h(\mb{x}) \end{equation*}
    Then
    \begin{equation*} \lim_{\mb{x} \to \mb{c}} g(\mb{x}) = L. \end{equation*}

    Example 1.1.13. Squeeze Theorem Application.

    The function
    \begin{equation*} f(x,y) = \frac{x^3 + y^3}{x^2 + y^2} \end{equation*}
    is continuous everywhere except at the origin where it is undefined. We can nonetheless consider the limit of \(f(x,y)\) as one approaches \((0, 0)\text{.}\) To see this limit exists, we first observe that \(f(x,y)\) satisfies the inequalities
    \begin{equation*} -2 \sqrt{x^2 + y^2} \leq f(x,y) \leq 2 \sqrt{x^2 + y^2 }. \end{equation*}
    To see this, just use polar coordinates \((x,y) = (r \cos \theta , r \sin \theta)\) and observe that
    \begin{equation*} f(x,y) = r \left( \cos^3 \theta + \sin^3 \theta \right). \end{equation*}
    Now, the function \(\sqrt{x^2 + y^2}\) is the composition of continuous functions, so it is continuous and thus the left and right hand bound have limit \(0\) as \((x,y) \to (0, 0)\text{.}\) But then the Squeeze Theorem gives that
    \begin{equation*} \lim_{(x,y) \to (0, 0)} f(x,y) = 0. \end{equation*}
    When considering examples of pathological behavior for limits in two dimensions, it is often useful to use polar coordinates. In other words, to consider functions \(f(r, \theta)\) and contemplate
    \begin{equation*} \lim_{r \to 0} f(r, \theta) \end{equation*}
    For example, it would be simple to observe that
    \begin{equation*} f(r, \theta) = \cos (2\theta ) \end{equation*}
    does not have a limit at the origin, but does have limits along any lines. Converting this to Cartesian coordinates, use the double angle formula to get
    \begin{equation*} f(r, \theta) =2 \cos^2 \theta - 1 = 2 \frac{x^2}{x^2 + y^2} - 1 = \frac{x^2 - y^2}{x^2 + y^2} \end{equation*}
    This can be illustrated using Sage.

    1.

    Show that a polynomial \(f(z) = a_n z^n + a_{n - 1} z^{n - 1} + \cdots + a_1 z + a_0\) viewed as a complex function from \(\mathbb{C}\) to itself is continuous.

    2.

    Find the limits of the following functions if they exist. If they do not exist, explain why.
    (a)
    \(f(x, y) = \frac{x^2y - y^2x}{x^2 + y^2}\) as \((x,y) \to (0,0)\text{.}\)
    (b)
    \(g(x,y,z) = \frac{xy + yz + xz}{x^2 + y^2 + z^2}\) as \((x,y,z) \to (0,0,0)\text{.}\)
    (c)
    \(k(x,y) = \frac{x^2 + 2y^2}{\sqrt{x^2 + 2y^2 +1} - 1}\) as \((x,y) \to (0,0)\text{.}\)
    (d)
    \(h(\mb{x}) = \frac{Q(\mb{x} )}{\|\mb{x}\|^2}\) as \(\mb{x} \to \mb{0}\) where \(Q\) is a non-zero quadratic form on a finite dimensional inner product space.

    3.

    Let \(\mathcal{B} = \{\mb{v}_1 , \ldots, \mb{v}_n\}\) be a basis for the real inner product space \(V\text{.}\)
    (a)
    Let \(A\) be the \(n \times n\) matrix with \((i,j)\)-entry \(\langle \mb{v}_i , \mb{v}_j \rangle\text{.}\) Show that the quadratic form
    \begin{equation*} Q (\mb{x}) = \mb{x}^T A \mb{x} \end{equation*}
    is positive definite (recall this means that \(Q(\mb{x} ) \geq 0\) with equality if and only if \(\mb{x} = \mb{0}\text{.}\))
    (b)
    Use the Spectral Theorem to show that there are positive real constants \(C\) and \(D\) for which
    \begin{equation*} C \| \mb{x}\|^2 \leq Q ( \mb{x} ) \leq D \| \mb{x} \|^2. \end{equation*}
    (c)
    Using the definition of limit, prove that the linear transformations \(\coord{}{\mathcal{B}} : V \to \mathbb{R}^n\) and \([]_{\mathcal{B}} : \mathbb{R}^n \to V\) are continuous functions.
