    Section 4.1 Integrating forms

    In this section, we consider changing both the integrand and the domain of integration to obtain an expression
    \begin{equation*} \int_{D} \eta \end{equation*}
    where \(D\) is a \(k\)-dimensional oriented domain in \(\mathbb{R}^m\) and \(\eta\) is a differential \(k\)-form. To make sense of this, we proceed in steps.

    Subsection 4.1.1 Integrating top forms

    Let \(D\) be a domain in \(\mathbb{R}^m\) which is the closure of an open set \(U\) and has a smooth boundary (perhaps with some corners).

    Definition 4.1.1.

    An orientation form on \(D\) is a nowhere vanishing differential \(m\)-form \(\vol\) for which \(\vol_p\) is a unit vector in \(\wedge^m T^*_p \mathbb{R}^m\) for each \(p \in D\text{.}\) We say that \(D\) is oriented if it is equipped with an orientation form. If \(\eta\) is any \(m\)-form on \(D\) then we say it is compatible with orientation if \(\eta = f (x) \vol\) for a strictly positive scalar function \(f(x)\text{.}\)
    The standard orientation is the form
    \begin{equation*} \vol^{st} = \tder{}{x_1} \wedge \cdots \wedge \tder{}{x_m} \end{equation*}
    The idea of an oriented domain in \(\mathbb{R}^m\) is that we ignore the coordinates for a moment and imagine equipping the domain with its own orientation. The following result can be found in a topology course. To state it, we remind the student that a domain \(D\) is connected if, given any two open sets \(U\) and \(V\) for which \(D\) is contained in \(U \cup V\text{,}\) we must have that \(D\) is contained in \(U\) or \(D\) is contained in \(V\text{.}\)

    Proposition 4.1.2.

    If \(D\) is open and connected and \(\vol\) is an orientation form on \(D\) then \(\vol = \pm \vol^{st}\text{.}\)
    Let’s digest this with \(m = 1\) and replace \(\mathbb{R}^1\) with \(V\text{.}\) We could imagine an interval \(I\) in \(V\) and hope to find the integral
    \begin{equation*} \int_I f \end{equation*}
    for some function \(f\) on \(I\text{.}\) If \(F\) is its anti-derivative, one could use the Fundamental Theorem of Calculus to get
    \begin{equation*} \int_I f = F(b) - F(a) \end{equation*}
    But of course this means we have \(I = [a, b]\) and in a sense, have oriented \(I\text{.}\) If we chose an opposite orientation, then we may obtain the opposite answer
    \begin{equation*} \int_I f = F(a) - F(b). \end{equation*}
    Indeed, we now can make a general definition.

    Definition 4.1.3.

    Suppose \((D, \vol)\) is an oriented domain in \(\mathbb{R}^m\) and \(\eta\) a differential \(m\)-form on \(D\text{.}\) Then there is a scalar function \(f(x)\) on \(D\) for which
    \begin{equation*} \eta = f(x) \vol. \end{equation*}
    Let
    \begin{equation*} \int_D \eta = \idotsint_D f(x_1, \ldots, x_n) \tder{}{V}. \end{equation*}
    If we omit the orientation, we assume that \(D \subseteq \mathbb{R}^m\) has the standard orientation. To calculate with orientations, we need to know when a parameterization of an oriented domain retains the correct orientation information.

    Definition 4.1.4.

    If \((D, \vol_D)\) and \((E, \vol_E)\) are oriented domains, and \(\mb{G} : D \to E\) is a change of variables, we say that \(\mb{G}\) preserves orientation if \(\mb{G}^* \vol_E\) is compatible with the orientation of \(D\text{.}\)
    With this in mind, we can write the differential form version of change of variables.

    Theorem 4.1.5. Chain Rule.

    Suppose \(D\) and \(E\) are oriented domains in \(\mathbb{R}^m\text{.}\) Let \(\mb{G} : D \to E\) be a change of variables which preserves orientation and \(\eta\) an \(m\)-form on \(E\text{.}\) Then
    \begin{equation*} \int_D \mb{G}^* \eta = \int_E \eta. \end{equation*}

    Proof.

    While it is certainly true that there is some mathematics buried within the notation of differential forms, orientations and pullbacks, the above form shows the drastic simplifying power of using forms to express complicated formulas. This, of course, is just the beginning of their utility.

    Subsection 4.1.2 Integrating \(k\)-forms on oriented subspaces.

    Recall from Definition 1.7.2 that a smooth \(m\)-dimensional subspace \(X\) of \(\mathbb{R}^n\) is any subspace with a parameterization \(\mb{G} : U \to X\) near every point \(p\) of \(X\text{.}\) We will need to add orientations to such subspaces as a part of their package.

    Definition 4.1.6.

    Suppose \(X\) is a smooth \(m\)-dimensional subspace of \(\mathbb{R}^n\text{.}\) An orientation on \(X\) is a differential \(m\)-form \(\orient^X\) defined on a neighborhood of \(X\) such that, for any local parameterization \(\mb{G}:U \to X\) the pullback \(\mb{G}^* \orient^X\) is nowhere zero on \(U\text{.}\) Two orientations are equivalent if their pullbacks are compatible for any local parameterization. We say \((X, \orient^X)\) is an oriented smooth \(m\)-dimensional subspace of \(\mathbb{R}^n\text{.}\)
    From this definition, the student may wonder if orientable smooth spaces are hard to find. The next proposition shows that, in fact, they are in ample supply.

    Proposition 4.1.7.

    Suppose \(f : \mathbb{R}^n \to \mathbb{R}\) is a smooth scalar function on Euclidean space for which \(c\) is not a critical value. Then the smooth level subspace
    \begin{equation*} X = \left\{ \mb{x} \in \mathbb{R}^n : f (\mb{x} ) = c \right\} \end{equation*}
    is orientable with orientation form
    \begin{equation*} \orient^X = \star \tder{}{f} \end{equation*}

    Proof.

    Some orientations are very familiar. To see this, we first need to use inner products to take vector fields to \(1\)-forms. Suppose \(X\) is a vector field in \(\mathbb{R}^n\text{.}\) Define the dual differential \(1\)-form \(X^\vee\) to be the field of cotangent vectors \(X^\vee_p\) which are defined by
    \begin{equation} X^\vee_p (Y) = X \cdot Y\tag{4.1.1} \end{equation}
    for any tangent vector \(Y \in T_p \mathbb{R}^n\text{.}\)

    Example 4.1.8. Oriented curve.

    If \(\mathcal{C}\) is a connected curve, then an orientation on \(\mathcal{C}\) is equivalent to a continuous choice of unit tangent vector of each point of \(\mathcal{C}\text{.}\) One can think of this as a vector field \(X\) along \(\mathcal{C}\text{.}\) To obtain an orientation as given in Definition 4.1.6, define the orientation form as the dual \(1\)-form \(X^\vee\text{.}\)

    Example 4.1.9. Oriented surface.

    If \(\mathcal{S}\) is a surface in \(\mathbb{R}^3\text{,}\) then one often defines an orientation on \(\mathcal{S}\) as a continuous choice of unit normal vector \(\mb{N}_p\) as \(p\) varies in \(\mathcal{S}\text{.}\) To get from this to Definition 4.1.6, take the Hodge star operator of the dual \(1\)-form \(\star (\mb{N}^\vee)\text{.}\)
    As it turns out, once the dimension of the space exceeds \(1\text{,}\) there are examples of subspaces that are not orientable. The Möbius strip is one such example in dimension \(2\text{.}\) One reason for orienting a smooth subspace is that for such spaces we can integrate forms.

    Definition 4.1.10.

    Let \((X, \orient^X)\) be a smooth oriented \(m\)-dimensional subspace of \(\mathbb{R}^n\) compatibly parameterized by \(\mb{G} : D \to X\text{.}\) If \(\eta\) is an \(m\)-form on a neighborhood of \(X\) then define
    \begin{equation*} \int_X \eta = \int_D \mb{G}^* \eta. \end{equation*}
    This definition makes sense because of the version of the chain rule.

    Lemma 4.1.11. Chain rule.

    Let \(D_1, D_2\) and \(D_3\) be domains in Euclidean spaces. Suppose \(\mb{G}_1 : D_1 \to D_2\) and \(\mb{G}_2 : D_2 \to D_3\) are differentiable functions and \(\eta\) is a differentiable \(k\)-form on \(D_3\text{,}\) then
    \begin{equation*} (\mb{G}_2 \circ \mb{G}_1)^* \eta = \mb{G}_1^* (\mb{G}_2^* \eta ). \end{equation*}

    Proof.

    Exercise.
    This lemma ensures our definition makes sense because, if \(\tilde{\mb{G}} : \tilde{D} \to X\) is another parameterization then we may compose with the inverse of \(\mb{G}\) to get a change of variables \(\mb{F} := \tilde{\mb{G}} \circ \mb{G}^{-1} : \tilde{D} \to D\) and, applying a change of variables we see
    \begin{equation*} \int_{\tilde{D}} \tilde{\mb{G}}^* \eta = \int_{\tilde{D}} \tilde{\mb{G}}^* \left( (\mb{G}^{-1})^* \left( \mb{G}^* \eta \right) \right) = \int_{\tilde{D}} \left(\tilde{\mb{G}} \circ \mb{G}^{-1} \right)^* \left( \mb{G}^* \eta \right) = \int_D \mb{G}^* \eta. \end{equation*}
    Thus the quantity obtained is independent of the parameterization.
    To extend Definition 4.1.10 to any smooth oriented \(m\)-dimensional subspace (not just ones with a single parameterization), one must consider a finite cover \(V_1, \ldots, V_r\) of \(X\) by open sets and with compatibly oriented parameterizations \(\mb{G}_i : U_i \to X\text{.}\) Then we define what is known as a partition of unity \(\{\rho_1, \ldots, \rho_r\}\) subordinate to the cover which is a collection of non-negative scalar functions \(\rho_i\) on \(X\text{.}\) Each \(\rho_i\) is only non-zero on \(V_i\) and their sum \(\sum \rho_i = 1\) is the constant function \(1\text{.}\) With such a partition of unity in hand, we can define the integration of an \(m\)-form on \(X\) to be
    \begin{equation*} \int_X \eta = \sum_{i = 1}^r \int_{U_i} \mb{G}_i^* (\rho_i \eta). \end{equation*}
    We will revisit partitions of unity with a bit more detail when we prove the Generalized Stokes’ Theorem.

    Write down four distinct orientations for the domain
    \begin{equation*} D = \{ (x,y) : x \ne 0 \}. \end{equation*}

    2.

    Using the change of coordinates \(\mb{G} (r, \theta) = (r \cos \theta, r \sin \theta)\text{,}\) find the pullback forms :
    (a)
    \(\mb{G}^* [(x^2 + y^2 )\tder{}{x} - y \tder{}{y} ]\text{,}\)
    (b)
    \(\mb{G}^* \frac{\tder{}{x} \wedge \tder{}{y}}{\sqrt{x^2 + y^2}}\)

    3.

    Prove the differential form version of the change of variables Theorem 4.1.5.

    4.

    Prove the claim in Example 4.1.9 that \(\star (\mb{N}^\vee)\) defines an orientation on a surface \(\mathcal{S}\) in \(\mathbb{R}^3\) if \(\mb{N}\) is a continuously normal vector field to \(\mathcal{S}\text{.}\)

    5.

    Prove the differential form version of the chain rule for \(1\)-forms \(\eta\text{.}\)

    6.

    Let \(\mathcal{C}\) be the ellipse given by
    \begin{equation*} x^2 + \frac{y^2}{4} = 1 \end{equation*}
    oriented counterclockwise.
    (a)
    Give a compatibly oriented and non-compatibly oriented parameterization of \(\mathcal{C}\text{.}\)
    (b)
    For the one form \(\eta = y \, \tder{}{x}\text{,}\) compute
    \begin{equation*} \int_\mathcal{C} \eta. \end{equation*}
    (c)
    For the one form \(\tau = 4 x \, \tder{}{x} + y \tder{}{y}\text{,}\) compute
    \begin{equation*} \int_\mathcal{C} \tau. \end{equation*}

    7.

    Let \(\mathcal{C}\) be a straight line curve in \(\mathbb{R}^5\) from the origin to \((1, 2, 3, 4, 5)\text{.}\) Compute
    \begin{equation*} \int_{\mathcal{C}} x_1 \tder{}{x_1} + \frac{x_2}{4} \tder{}{x_2} + \frac{x_3}{9} \tder{}{x_3} + \frac{x_4}{16} \tder{}{x_4} + \frac{x_5}{25} \tder{}{x_5}. \end{equation*}

    8.

    Prove Proposition 4.1.7.

    9.

    Let \(\mathcal{S}\) be the unit sphere in \(\mathbb{R}^3\) oriented with outward normal.
    (a)
    Give an example of a \(2\)-form in \(\mathbb{R}^3\) that induces an orientation on \(\mathcal{S}\text{.}\)

    Compute the surface integral
